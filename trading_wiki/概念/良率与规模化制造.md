# 良率与规模化制造

## 一句话解释

良率是合格产品占总产出的比例。规模化制造的核心不是做出一个样品，而是稳定、低成本、大批量地做出合格产品。

## 从样品到量产

| 阶段 | 关键问题 |
| --- | --- |
| 实验室样品 | 原理是否可行 |
| 工程样机 | 设计是否可制造 |
| 小批量试产 | 工艺是否稳定 |
| 大规模量产 | 良率、成本、供应链是否可控 |
| 持续改进 | 质量和效率能否提升 |

## 良率影响

| 影响 | 说明 |
| --- | --- |
| 成本 | 报废越多，单位成本越高 |
| 交付 | 良率不稳会影响客户订单 |
| 利润率 | 良率提升常直接改善毛利 |
| 认证 | 客户需要稳定品质 |
| 竞争壁垒 | 工艺经验难以快速复制 |

## 我的启发

制造业机会不能只听“技术突破”，还要看良率、产能、客户认证、设备瓶颈、材料供应和成本曲线。
