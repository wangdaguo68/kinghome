# 周期资源与商品价格传导地图

## 概念定义

周期资源与商品价格传导地图，是把宏观需求、供给扰动、库存周期、美元利率、地缘风险、运输成本和企业利润弹性串起来的专题地图。周期股不是看“价格涨”一个变量，而要看价格上涨是否能穿透成本、库存和产量，最终进入利润。

## 价格传导框架

| 步骤 | 问题 | 反证 |
| --- | --- | --- |
| 需求 | 是真实需求上升，还是补库/情绪 | 终端开工不配合 |
| 供给 | 是供给收缩，还是短期扰动 | 产能快速恢复 |
| 库存 | 显性和隐性库存是否下降 | 库存累积压制价格 |
| 成本 | 成本是否同步上涨 | 价格涨但利润不涨 |
| 公司弹性 | 公司有产量、资源和成本优势吗 | 贸易商/加工商弹性弱 |
| 估值 | 股价是否已经反映价格预期 | 商品回落杀估值 |

## 验证清单

- [ ] 商品价格是现货、期货、长协还是指数。
- [ ] 公司是资源端、冶炼端、加工端还是贸易端。
- [ ] 价格上涨是否能进入利润，成本是否同步上涨。
- [ ] 库存和开工率是否支持价格持续。
- [ ] 地缘扰动是否已经发生，持续时间多长。

## 结论

周期资源专题必须同时看“商品价格”和“公司利润弹性”。价格上涨只是第一层，真正的交易认知在供需、库存、成本、产量、税费、汇率和估值。
