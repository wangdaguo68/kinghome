# 审美心理学

## 概念定义

审美心理学研究人如何感知、评价和体验美。它关注形式、秩序、新奇、复杂度、熟悉感、情绪、记忆和文化经验如何共同形成“好看”“高级”“有味道”“不舒服”等判断。

## 审美体验的组成

| 组成 | 解释 | 例子 |
| --- | --- | --- |
| 感知流畅性 | 越容易被组织和理解，越容易产生愉悦 | 清晰排版、稳定比例 |
| 适度复杂 | 太简单无聊，太复杂混乱 | 有细节但不噪音 |
| 新奇与熟悉 | 既有识别基础，又有一点意外 | 经典款式的新材质 |
| 情绪唤起 | 形式会触发安静、兴奋、温暖、庄重 | 色彩、音乐、空间尺度 |
| 文化学习 | 审美偏好会被时代、阶层、媒介训练 | 极简、国潮、复古 |
| 身份投射 | 人会把审美选择当成自我表达 | 穿搭、家居、品牌选择 |

## 审美不是纯主观

| 层面 | 说明 |
| --- | --- |
| 生理层 | 感知、对称、对比、节奏、颜色会影响体验 |
| 认知层 | 信息是否易理解，是否形成有意义结构 |
| 情绪层 | 是否带来愉悦、惊奇、安全或崇高感 |
| 文化层 | 社会教会我们什么叫高级、俗气、先锋 |
| 个人层 | 记忆、身份、经历塑造偏好 |

## 训练方法

1. 建立样本库：收集你认为好的设计、品牌、空间、影像。
2. 拆解原因：不是只说“好看”，而是写清比例、色彩、节奏、材质、语气、场景。
3. 做反例对比：找同类中失败案例，比较差异。
4. 建立风格词表：极简、秩序、温润、锋利、克制、复古、未来感。
5. 连接体验：问它是否让人更好理解、更愿意行动、更愿意记住。
