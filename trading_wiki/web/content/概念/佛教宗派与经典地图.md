# 佛教宗派与经典地图

## 概念定义

佛教宗派是不同历史、语言、经典和修行传统对佛陀教法的展开。宗派差异不等于互相否定，更多是重点、方法和解释体系不同。

## 三大传统

| 传统 | 经典语言和文本 | 重点 | 相关页面 |
| --- | --- | --- | --- |
| 南传上座部 | 巴利三藏 | 四圣谛、八正道、戒定慧、内观、阿毗达摩 | [[南传上座部与阿毗达摩]] |
| 汉传佛教 | 汉译大藏经 | 大乘经典、禅、净土、天台、华严、律 | [[中国佛教禅宗净土天台华严]] |
| 藏传佛教 | 藏文大藏经 | 中观、瑜伽行、密续、次第道、上师传承 | [[菩萨道与六度]]、[[空性与二谛]] |

## 重要思想流派

| 流派 | 核心关注 | 易误读点 |
| --- | --- | --- |
| 中观 | 缘起性空、破除自性执 | 空不是虚无 |
| 瑜伽行/唯识 | 识、表象、种子、认知构造 | 唯识不是简单唯心主义 |
| 天台 | 一心三观、法华思想、判教 | 不只是理论分类 |
| 华严 | 法界缘起、事事无碍 | 不等于玄学想象 |
| 禅宗 | 直指人心、见性、坐禅和日常实践 | 不是任性反智 |
| 净土 | 信愿行、念佛、往生净土 | 不是单纯求保佑 |
| 密教 | 仪轨、观想、咒、上师传承 | 不能脱离传承和戒律 |

## 经典地图

| 类型 | 代表经典 | 学习用途 |
| --- | --- | --- |
| 早期经典 | 阿含经、尼柯耶 | 理解四圣谛、缘起、八正道 |
| 般若类 | 心经、金刚经、大般若经 | 理解空性和无所得 |
| 法华类 | 法华经 | 理解一乘、方便、菩萨道 |
| 华严类 | 华严经 | 理解法界缘起和菩萨境界 |
| 净土类 | 无量寿经、观无量寿经、阿弥陀经 | 理解净土信愿行 |
| 禅宗文本 | 六祖坛经、景德传灯录 | 理解汉传禅宗语言和实践 |
| 律典 | 四分律等 | 理解僧团戒律 |
| 论典 | 中论、俱舍论、成唯识论 | 理解哲学分析和心理分类 |

## 初学路径

1. 先读[[四圣谛与八正道]]、[[缘起与十二因缘]]、[[三法印与无我]]。
2. 再根据兴趣选择南传、汉传或大乘哲学。
3. 读经典时先看可靠导读，再读原文。
4. 分清“经典原文、传统注疏、现代解释、个人感悟”四个层级。

## 常见误读

| 误读 | 更准确理解 |
| --- | --- |
| 宗派越神秘越高级 | 可靠传统都重视戒、定、慧和慈悲 |
| 禅宗不需要经典 | 禅宗历史上与经典和戒律有复杂关系，不是反知识 |
| 净土只是迷信 | 净土有信愿行、愿力、念佛实践和大乘背景 |
| 密教可以自学乱修 | 密教尤其重视传承、戒律和指导 |
