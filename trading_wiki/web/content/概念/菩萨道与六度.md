# 菩萨道与六度

## 概念定义

菩萨道是大乘佛教中以觉悟和利益众生为方向的修行道路。六度，也叫六波罗蜜，是布施、持戒、忍辱、精进、禅定、般若。

| 六度 | 简明解释 | 对治 |
| --- | --- | --- |
| 布施 | 给予、分享、放下占有 | 贪 |
| 持戒 | 行为和关系的清净 | 混乱和伤害 |
| 忍辱 | 面对逆境不被嗔恨吞没 | 嗔 |
| 精进 | 持续向善和修行 | 懈怠 |
| 禅定 | 稳定和清明 | 散乱 |
| 般若 | 空性智慧 | 痴和自性执 |

## 菩萨道的核心：智慧与慈悲

只有慈悲没有智慧，容易变成情绪化拯救；只有智慧没有慈悲，容易变成冷漠的概念游戏。

| 偏差 | 表现 | 修正 |
| --- | --- | --- |
| 慈悲无智慧 | 过度牺牲、讨好、边界混乱 | 用般若和戒建立边界 |
| 智慧无慈悲 | 讲空性但缺少温度 | 用布施和忍辱训练关系 |
| 精进无定 | 忙碌、焦躁、求功德 | 回到正念和稳定 |
| 禅定无慧 | 贪着宁静体验 | 继续观察无常和无我 |

## 菩提心

菩提心通常指为了利益众生而求觉悟的心。它不是宏大口号，而是把“我怎样更自由”扩展成“我如何减少自己和他人的痛苦”。

## 现代生活中的六度

| 六度 | 可执行观察 |
| --- | --- |
| 布施 | 是否能分享时间、资源、注意力，而不是只索取 |
| 持戒 | 是否减少欺骗、伤害和成瘾行为 |
| 忍辱 | 被冒犯时能否先观察受伤感，而不是立刻攻击 |
| 精进 | 是否稳定做正确的小事 |
| 禅定 | 是否每天有一点不被信息流拖走的时间 |
| 般若 | 是否能看见自我故事和条件网络 |

## 常见误读

| 误读 | 更准确理解 |
| --- | --- |
| 慈悲就是无条件满足别人 | 慈悲需要智慧和边界 |
| 忍辱就是忍气吞声 | 忍辱是心不被嗔恨支配，不等于纵容伤害 |
| 布施就是捐钱 | 布施包括财物、知识、无畏和善意 |
| 菩萨道比个人修行高级所以可以跳过基础 | 菩萨道仍离不开戒定慧 |
