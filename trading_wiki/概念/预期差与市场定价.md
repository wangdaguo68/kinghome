# 预期差与市场定价

## 一句话解释

市场交易的不是事实本身，而是事实相对预期的变化。预期差是“真实变化”与“市场已经相信的东西”之间的差。

## 三层判断

| 层级 | 问题 |
| --- | --- |
| 事实 | 发生了什么 |
| 预期 | 市场原本以为会怎样 |
| 定价 | 当前价格反映了多少 |

## 典型情况

| 情况 | 可能结果 |
| --- | --- |
| 好消息低于高预期 | 下跌 |
| 普通消息高于低预期 | 上涨 |
| 利好已充分传播 | 冲高回落 |
| 反证被忽略 | 后续补跌 |

## 我的启发

研究不能只问“逻辑对不对”，还要问“别人是否已经知道、相信并买入”。没有预期差的正确观点，也可能不是好机会。
