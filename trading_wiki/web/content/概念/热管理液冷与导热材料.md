# 热管理液冷与导热材料

## 一句话解释

热管理是把电子设备和数据中心产生的热量及时带走，避免性能下降和寿命缩短。AI 高功耗集群推动液冷和导热材料重要性上升。

## 散热路径

芯片发热 -> 封装材料 -> 导热界面材料 -> 冷板/散热器 -> 冷却液/空气 -> 机房冷却系统

## 关键技术

| 技术 | 作用 |
| --- | --- |
| 导热界面材料 | 降低芯片和散热器之间热阻 |
| 冷板 | 把热量传给冷却液 |
| 液冷 | 用液体带走高密度热量 |
| 浸没式冷却 | 设备浸入绝缘冷却液 |
| 相变材料 | 利用相变吸收或释放热量 |

## 我的启发

算力密度越高，散热越从配套环节变成核心约束。产业研究时要看客户验证、可靠性、材料认证和规模交付能力。

## 2026-06-15 认知更新：补入氟化液冷

新增[[制冷剂与氟化液冷]]作为热管理材料下钻。热管理需要同时看：

| 层级 | 代表品类 | 关键验证 |
| --- | --- | --- |
| 导热材料 | TIM、导热垫片、导热凝胶、石墨片 | 芯片/服务器客户和单机价值量 |
| 液冷设备 | 冷板、CDU、泵阀、管路 | 数据中心订单和毛利 |
| 冷却介质 | 水基冷却液、氟化液、浸没液冷介质 | 绝缘、热稳定、环保、客户认证 |
