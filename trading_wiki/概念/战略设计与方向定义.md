# 战略设计与方向定义

## 概念定义

战略设计是把未来判断、系统理解、用户需求、组织能力和价值选择整合成方向。战略不是愿景口号，也不是任务清单，而是选择在哪里竞争、如何取舍、用什么系统达成目标。

## 战略的核心结构

| 组成 | 问题 |
| --- | --- |
| 诊断 | 当前局势的关键矛盾是什么 |
| 方向 | 我们要创造什么不同未来 |
| 选择 | 做什么，不做什么 |
| 能力 | 需要哪些核心能力 |
| 系统 | 组织、流程、资源如何配合 |
| 反馈 | 如何知道方向是否正确 |

## 战略和计划的区别

| 项目 | 战略 | 计划 |
| --- | --- | --- |
| 关注 | 方向和取舍 | 任务和时间表 |
| 面对 | 不确定性和竞争 | 已知工作 |
| 输出 | 核心选择和资源配置 | 项目清单 |
| 失败 | 方向错或没有取舍 | 执行延误 |

## 主动定义新方向

| 方法 | 说明 |
| --- | --- |
| 重定义问题 | 不在旧问题上更努力，而是改变问题框架 |
| 找系统杠杆点 | 抓住能改变反馈回路的位置 |
| 构建新类别 | 让用户用新分类理解你 |
| 设计新体验 | 用体验证明新方向可行 |
| 建立新叙事 | 让组织和用户知道为什么要转向 |

## 对个人知识库的用法

个人战略不是“我要学很多”，而是确定你要成为怎样的复合能力系统。九大系统可以作为长期战略框架：认知负责理解人，智能理解 AI，产业和财富理解现实价值，美学和意义理解文化，范式理解未来。
