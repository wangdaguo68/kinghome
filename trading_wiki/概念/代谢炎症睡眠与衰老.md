# 代谢炎症睡眠与衰老

## 一句话解释

代谢决定身体如何获取和使用能量，炎症是免疫反应的一部分，睡眠是修复和调节系统，衰老则是多种损伤、调控和适应变化长期累积的结果。

## 核心变量

| 变量 | 影响 |
| --- | --- |
| 胰岛素 | 血糖调节和能量储存 |
| 线粒体 | 细胞能量生产 |
| 炎症 | 防御和修复，慢性化后会损害健康 |
| 昼夜节律 | 睡眠、激素、代谢和认知 |
| 肌肉量 | 代谢健康、运动能力和衰老质量 |
| 睡眠质量 | 情绪、免疫、记忆和修复 |

## 我的启发

健康不是单一指标，而是睡眠、运动、饮食、压力、代谢和炎症的系统结果。不要被单个补剂或单个指标叙事带走。

## 风险和误区

- 不要把长寿科学简化成吃某种药。
- 不要用单次体检指标替代长期趋势。
- 不要把睡眠牺牲当成努力。
