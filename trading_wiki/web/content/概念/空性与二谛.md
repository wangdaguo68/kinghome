# 空性与二谛

## 概念定义

空性是大乘佛教尤其是中观思想的核心概念，指一切法没有独立、固定、自足的自性。二谛是世俗谛和胜义谛：在日常约定层面事物有效存在，在究竟观察中找不到独立自性。

最重要的一句话：空不是没有，而是无自性。

## 空性和缘起

[[缘起与十二因缘]]说明一切依赖条件而生。既然依赖条件，就不是独立自存；既然不是独立自存，所以说空。

| 角度 | 说明 |
| --- | --- |
| 缘起 | 从条件网络看现象如何成立 |
| 空性 | 从无自性看现象不可被固定化 |
| 二者关系 | 因为缘起，所以性空；因为性空，所以可变 |

## 二谛

| 二谛 | 含义 | 例子 |
| --- | --- | --- |
| 世俗谛 | 日常语言、关系、因果中成立 | “我在写笔记”“这是一张桌子” |
| 胜义谛 | 究竟分析中无固定自性 | “我”和“桌子”都依赖部分、概念、功能和关系 |

二谛的价值是避免两种极端：执实和断灭。

## 用例：自我评价

“我是失败者”在世俗层面是一个念头和评价；在胜义分析中，它依赖某次结果、记忆、比较对象、身体情绪、语言标签和他人反馈，不是一个固定实体。

看见空性，不是说失败不存在，而是说“失败者这个固定自我”不可得。

## 与中观、唯识的关系

| 学派 | 重点 |
| --- | --- |
| 中观 | 破除一切自性执，强调缘起性空 |
| 唯识/瑜伽行 | 分析识、表象、种子和认知建构 |

两者都深刻影响大乘佛教，但解释路径不同，不能混为一谈。详见[[佛教宗派与经典地图]]。

## 常见误读

| 误读 | 更准确理解 |
| --- | --- |
| 空性就是不存在 | 空性是否定自性，不是否定现象 |
| 懂空性就不用努力 | 正因为因缘有效，行为才有后果 |
| 空性可以用来逃避责任 | 这是把胜义谛误用到世俗伦理 |
| 一切都是幻觉 | 更准确是现象依条件和认知成立，不是完全没有 |
