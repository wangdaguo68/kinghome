# 哲学基础地图

## 一句话解释

哲学是对存在、知识、价值、意义、语言、心灵和行动的根本追问。它不是背诵观点，而是训练一个人如何把模糊问题拆成可讨论、可反思、可实践的结构。

本页归入[[意义系统全景地图]]，关注个人如何理解世界、自己和价值。与[[科学哲学与知识更新]]的区别是：科学哲学主要研究科学知识如何形成和更新；本页更关注生活意义、伦理选择、存在体验和思想传统。

## 核心分支

| 分支 | 追问 | 典型问题 |
| --- | --- | --- |
| 形而上学 | 什么是真实存在 | 世界是物质、心灵、过程，还是关系 |
| 认识论 | 人如何知道 | 知识、信念、证据、怀疑和确定性 |
| 伦理学 | 什么是应该 | 善、责任、幸福、德性、后果和义务 |
| 政治哲学 | 人如何共同生活 | 自由、平等、权力、正义、共同体 |
| 美学 | 什么是美 | 审美经验、形式、风格、艺术价值 |
| 语言哲学 | 语言如何承载意义 | 概念、命名、隐喻、解释和误解 |
| 心灵哲学 | 意识和自我是什么 | 自由意志、意识、身份、身心关系 |
| 存在主义 | 人如何面对自由和荒诞 | 选择、焦虑、责任、真实生活 |

## 学习路径

1. 先理解问题，而不是先背人物。
2. 从四个问题开始：我能知道什么、我应该做什么、我能希望什么、人是什么。
3. 进入[[哲学大师经典思想地图]]，看不同思想家分别在回答什么问题。
4. 每读一个哲学观点，都写清它在反对什么。
5. 把哲学观点和真实生活连接：选择、关系、消费、工作、交易、痛苦和意义。
6. 遇到抽象概念时，用一个现实案例重写。

## 关键概念

| 概念 | 通俗解释 | 现实用途 |
| --- | --- | --- |
| 存在 | 什么东西真正算作存在 | 避免把标签、叙事和真实混为一谈 |
| 知识 | 被证成的真信念或更复杂的可靠认知 | 区分知道、相信、猜测和转述 |
| 价值 | 人认为值得追求和维护的东西 | 判断什么值得投入时间和注意力 |
| 自由 | 能否选择，以及选择是否有责任 | 处理自律、选择、后悔和责任 |
| 意义 | 事件与人生整体之间的关系 | 面对挫折、重复劳动和长期目标 |
| 德性 | 一个稳定而值得培养的品质 | 建立长期人格和行为一致性 |
| 荒诞 | 人追问意义，但世界不直接给答案 | 面对不确定时代和个人困境 |
| 解释 | 用结构说明为什么如此 | 提升研究、表达和判断质量 |

## 我的启发

哲学的价值不在于给每个问题一个标准答案，而在于让人知道自己正在使用什么前提、概念和价值排序。它能帮助个人知识库避免只堆资料，而不反思“我为什么相信、为什么在意、为什么行动”。
