# 材料强度高分子与电子材料

## 一句话解释

材料决定产品能否承受力、热、电、化学和加工过程。AI硬件、半导体、机器人和汽车电子都离不开材料性能。

## 材料关键性能

| 性能 | 影响 |
| --- | --- |
| 强度 | 能否承受载荷和冲击 |
| 导电性 | 是否适合电路和连接 |
| 导热性 | 能否散热 |
| 介电性能 | 高频高速信号损耗 |
| 耐热性 | 能否承受工艺温度 |
| 加工性 | 是否能稳定量产 |
| 可靠性 | 长期使用是否失效 |

## 电子材料例子

| 材料 | 应用 |
| --- | --- |
| ABF载板 | 高端芯片封装 |
| 电子布 | PCB基材 |
| 铜箔 | PCB和锂电 |
| 玻璃基板 | 先进封装潜在方向 |
| PPE/PPO树脂 | 高频高速材料 |
| 电子特气 | 半导体制造 |

## 我的启发

材料研究不能只看“涨价”，要看材料在产品中的不可替代性、认证周期、客户结构、产能扩张难度和成本占比。
