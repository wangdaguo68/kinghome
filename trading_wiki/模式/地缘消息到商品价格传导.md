# 地缘消息到商品价格传导

## 模式定义

地缘消息只有在传导到商品价格、运价、保险费、供应链或政策反应后，才有产业链研究价值。

## 传导链

```text
地缘消息 -> 权威确认 -> 商品/运价反应 -> 成本或供给变化 -> 公司利润影响
```

## 观察指标

- Reuters/AP/官方声明是否确认。
- Brent/WTI、VLCC运价、保险费是否变化。
- 航线是否绕行或停运。
- 公司是受益还是受损。
