# 默认模式网络与自我叙事

## 概念定义

默认模式网络 DMN 是一组在安静休息、心智游走、自我相关思考、回忆过去、想象未来和理解他人心理时常被观察到的脑网络。

易懂理解：当你不专注于外部任务时，大脑常会自动回到“我是谁、我经历了什么、别人怎么看我、未来会怎样”的内在叙事模式。

## 主要相关区域

| 区域 | 功能倾向 |
| --- | --- |
| 内侧前额叶 | 自我相关评价、社会认知 |
| 后扣带/楔前叶 | 自我参照、内在注意 |
| 角回/顶下小叶 | 语义整合、情境想象 |
| 海马系统 | 记忆、情境和未来模拟 |

这些区域不是孤立工作，而是组成网络。

## 正常功能

默认模式网络不是坏东西。它有重要作用：

1. 整合个人经历。
2. 回忆过去。
3. 想象未来。
4. 理解他人心理。
5. 构建人生叙事。
6. 在任务间隙进行内在整理。

问题不在于有自我叙事，而在于叙事失控、反复、僵化，并与情绪系统互相放大。

## 反刍与内耗

| 现象 | 可能机制 |
| --- | --- |
| 反复想过去的错误 | 自我叙事和情绪记忆反复激活 |
| 担心未来失败 | 未来模拟和威胁系统耦合 |
| 总在想别人怎么看我 | 社会评价与自我参照增强 |
| 无法进入任务 | 默认模式网络和执行网络切换困难 |

## 与冥想和觉察的关系

冥想不是简单让大脑空白。很多练习是在训练：

1. 觉察注意跑到哪里。
2. 识别内在叙事只是心理事件。
3. 把注意带回呼吸、身体或当下任务。
4. 降低自动反刍的支配力。

这与[[注意力系统与执行控制]]、[[情绪系统与杏仁核]]和未来的佛学笔记可以连接。

## 常见误区

| 误区 | 更准确说法 |
| --- | --- |
| 默认模式网络就是坏的 | 它参与自我、记忆、想象和社会认知 |
| 冥想就是关闭 DMN | 研究结果更复杂，取决于练习类型和任务 |
| 自我叙事就是虚假 | 自我叙事有功能，但不等于全部现实 |
| 反刍只靠想通就能停 | 还要看情绪、身体状态、注意训练和环境 |
