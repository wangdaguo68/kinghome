# 六氟化钨WF6电子特气

## 概念定义

六氟化钨（WF6）是半导体制造中用于钨沉积的关键电子特气，常用于存储、逻辑芯片、先进制程中的钨栅、接触孔、互连等工艺。交易上关注 WF6，是因为先进制程、存储景气、国产替代和海外供给扰动可能共同影响价格与国产供应商订单。

本页不是“电子特气”泛化页，只讨论 WF6 及其直接产业链。

## 2026-06-15 认知更新：加入反向证据优先级

WF6仍然属于[[AI半导体材料与原料全景]]中的电子特气节点，但今日处理要更谨慎：如果出现公司澄清、收入占比低、客户未验证、价格未证实等反向证据，优先降低交易权限。

| 需要核验 | 为什么重要 |
| --- | --- |
| 公司是否真实生产/销售WF6 | 不能只看电子特气标签 |
| WF6收入占比 | 小收入占比不能支撑大市值弹性 |
| 客户和认证 | 半导体气体需要长周期认证 |
| 价格是否真实上涨 | 报价、合同价、成交价不同 |
| 产能是否可售 | 有产能不等于有客户和订单 |

## 结论

WF6 的交易价值在于“价格是否涨、合同是否穿透、客户是否认证、产能是否可售”。只看电子特气标签或涨停摘要，不能升级。
