# 五蕴六处十八界

## 概念定义

五蕴、六处、十八界是佛学拆解经验的工具。它们把“我正在经历什么”拆成可观察组件，帮助人看清自我感并不是一个固定实体。

## 五蕴

| 五蕴 | 简明解释 | 例子 |
| --- | --- | --- |
| 色 | 身体和物质层面 | 身体紧绷、呼吸、视觉声音 |
| 受 | 感受基调 | 苦受、乐受、不苦不乐受 |
| 想 | 识别和标记 | “这是批评”“这是机会” |
| 行 | 意志、习气、心理造作 | 想反击、想逃避、自动比较 |
| 识 | 分别认知 | 眼识、耳识、意识等识别活动 |

五蕴最重要的用途是观察“我”如何被拼装出来。例如“我被冒犯了”可以拆成：色是胸口紧和脸热，受是不舒服，想是把对方一句话标记成轻视，行是想辩解或攻击，识是持续把注意力放在这件事上。

## 六处

| 六根 | 对应对象 | 对应意识 |
| --- | --- | --- |
| 眼 | 色 | 眼识 |
| 耳 | 声 | 耳识 |
| 鼻 | 香 | 鼻识 |
| 舌 | 味 | 舌识 |
| 身 | 触 | 身识 |
| 意 | 法，包含念头、记忆、想象 | 意识 |

佛学把“念头”也当作经验对象。念头不是天然事实，而是被意识观察到的对象。

## 十八界

十八界是六根、六尘、六识合起来的经验结构。

| 结构 | 含义 | 用处 |
| --- | --- | --- |
| 六根 | 感知能力 | 经验入口 |
| 六尘 | 被感知对象 | 外界和内在对象 |
| 六识 | 对象被识别的意识 | 分别活动 |

## 与脑科学的连接

五蕴不是现代神经科学分类，但它与[[感知系统与身体内感]]、[[注意力系统与执行控制]]有对话价值：二者都提醒我们，经验不是直接复制外部世界，而是经过身体、感知、注意和解释共同生成。

## 常见误读

| 误读 | 更准确理解 |
| --- | --- |
| 五蕴是灵魂组成部分 | 五蕴是经验分析工具，不是固定实体清单 |
| 受就是情绪 | 受更基础，是愉快、不愉快、中性的感受基调 |
| 想就是思考 | 想偏向识别、标记和概念化 |
| 行就是行动 | 行包括意志、冲动、习气和心理造作 |
