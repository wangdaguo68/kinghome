# 机器人BOM电机减速器传感器

## 一句话解释

机器人 BOM 是机器人硬件成本和能力的拆解表。电机提供动力，减速器调节力矩和速度，传感器感知环境和自身状态，控制器协调执行。

## 核心部件

| 部件 | 作用 | 关键指标 |
| --- | --- | --- |
| 电机 | 产生运动 | 功率密度、效率、响应 |
| 减速器 | 提高力矩、降低转速 | 精度、寿命、回差 |
| 传感器 | 感知位置、力、视觉、惯性 | 精度、稳定性、成本 |
| 控制器 | 运动控制和协调 | 实时性、算法、可靠性 |
| 结构件 | 支撑和连接 | 强度、重量、加工精度 |
| 电池/电源 | 供能 | 能量密度、安全 |

## 产业判断

机器人能否量产，不只看 AI 模型，也要看硬件成本、可靠性、供应链、良率、维修和真实场景需求。
