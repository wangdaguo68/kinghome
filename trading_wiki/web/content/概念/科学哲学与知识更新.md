# 科学哲学与知识更新

## 概念定义

科学哲学研究科学知识如何形成、验证、修正和替代。它关注证据、解释、模型、因果、可证伪性、理论负载观察和科学共同体。

## 知识更新的基本链条

| 阶段 | 问题 | 风险 |
| --- | --- | --- |
| 观察 | 发生了什么 | 观察受理论和工具影响 |
| 假设 | 可能为什么发生 | 过早相信单一解释 |
| 预测 | 如果假设正确，应看到什么 | 只做事后解释 |
| 检验 | 证据是否支持 | 样本偏差、确认偏误 |
| 修正 | 旧模型哪里错了 | 不愿承认错误 |
| 共同体评议 | 他人能否复核 | 权威、利益和路径依赖 |

## 关键概念

| 概念 | 简明解释 |
| --- | --- |
| 可证伪性 | 一个理论应有可能被经验反驳 |
| 理论负载观察 | 观察不是纯中立，会受已有理论影响 |
| 模型 | 对现实的简化，不是现实本身 |
| 解释力 | 能否解释已知事实和异常现象 |
| 预测力 | 能否提前给出可检验判断 |
| 证据等级 | 不同证据强度不同，不能混为一谈 |

## 对个人知识库的用法

| 复盘问题 | 用途 |
| --- | --- |
| 这是事实、模型、假设还是观点？ | 防止混淆层级 |
| 什么证据会让我改变看法？ | 防止信念封闭 |
| 有没有更简单的替代解释？ | 防止过度叙事 |
| 这套解释能预测什么？ | 区分解释和预测 |
| 我是否只收集支持我的证据？ | 对抗确认偏误 |
