# 快速开始指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 运行方式

### 方式一:交互式菜单(推荐)

```bash
python main.py
```

然后按提示选择:
- 选项 1: 下载沪深300数据(首次使用必须执行)
- 选项 2: 单只股票回测
- 选项 3: 批量回测(测试前10只)

### 方式二:独立运行各模块

```bash
# 步骤1: 下载数据
python data.py

# 步骤2: 测试策略
python strategy.py

# 步骤3: 运行回测
python engine.py
```

## 3. 回测参数说明

- **初始资金**: 100,000元
- **手续费**: 0.03% (万分之三)
- **印花税**: 0.1% (千分之一,仅卖出)
- **策略**: 双均线 MA20/MA60

## 4. 输出指标

- 总收益率
- 年化收益率
- 最大回撤
- 交易次数
- 交易明细

## 5. 自定义开发

### 修改策略参数

编辑 `strategy.py` 中的 `dual_ma_strategy` 函数:

```python
dual_ma_strategy(df, short_window=20, long_window=60)
```

### 修改回测参数

编辑 `engine.py` 中的 `BacktestEngine` 初始化:

```python
BacktestEngine(initial_capital=100000, commission_rate=0.0003, stamp_tax=0.001)
```

## 注意事项

1. 首次运行需要下载约300只股票的5年数据,耗时约10-30分钟
2. 数据保存在 `data/hs300_5y.parquet` 文件中
3. akshare 数据源可能受网络影响,如失败请重试
