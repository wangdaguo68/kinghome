# 细胞基因蛋白质与免疫系统

## 一句话解释

细胞是生命基本单位，基因保存遗传信息，RNA参与信息传递和调控，蛋白质执行大量生命功能，免疫系统负责识别和应对威胁。

## 核心关系

DNA -> RNA -> 蛋白质 -> 细胞功能 -> 组织器官 -> 身体状态

| 概念 | 简明解释 |
| --- | --- |
| 细胞 | 生命活动基本单位 |
| DNA | 遗传信息载体 |
| RNA | 信息传递、调控和部分功能执行 |
| 蛋白质 | 酶、受体、结构、信号等功能分子 |
| 免疫系统 | 识别自身和非自身，清除病原和异常细胞 |

## 生物科技连接

| 技术 | 作用 |
| --- | --- |
| 基因测序 | 读取遗传信息 |
| mRNA技术 | 利用RNA表达目标蛋白 |
| 抗体药 | 精准识别靶点 |
| 细胞治疗 | 改造或利用细胞治疗疾病 |
| 蛋白质设计 | 设计具有特定功能的分子 |
