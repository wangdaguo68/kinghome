# 供需价格弹性与周期

## 一句话解释

价格由供需关系决定，但涨价能否变成利润，还取决于供给弹性、需求弹性、库存、成本和竞争格局。

## 核心原理

| 变量 | 问题 |
| --- | --- |
| 需求 | 是短期脉冲还是长期增长？ |
| 供给 | 能否快速扩产？ |
| 库存 | 渠道和企业库存高不高？ |
| 替代品 | 涨价后客户能否换方案？ |
| 成本 | 原材料涨价会不会吞掉利润？ |
| 竞争 | 行业内是否能共同提价？ |

## 周期链条

需求上升 -> 库存下降 -> 价格上涨 -> 利润改善 -> 资本开支增加 -> 产能释放 -> 价格回落

周期判断的关键是站在哪一段，不是只看到“涨价”两个字。

## 我的启发

涨价逻辑必须问三件事：涨价是否真实、公司能否传导、持续时间够不够长。不能把短期报价上涨直接当成业绩弹性。
