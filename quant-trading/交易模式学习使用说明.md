# 交易模式学习功能使用说明

## 功能概述

这个功能可以从你的历史成交记录中学习你的交易模式，通过机器学习模型预测你在特定行情形态下的买卖倾向。

## 核心优势

1. **个性化学习** - 基于你的实际交易行为，而非通用策略
2. **分钟级精度** - 关联成交时刻的分钟行情数据
3. **技术指标融合** - 综合MA、RSI、成交量等多维度特征
4. **置信度评估** - 提供预测的可信度参考

## 使用步骤

### 1. 准备历史成交记录

创建 `trade_history.xlsx` 文件，包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 股票代码 | 6位代码+市场后缀 | 000001.SZ, 600519.SH |
| 交易方向 | 买入或卖出 | 买入, 卖出 |
| 成交时间 | 精确到分钟 | 2026-01-15 10:30:00 |
| 成交价格 | 实际成交价 | 15.50 |
| 成交量 | 成交股数 | 1000 |

参考 `trade_history_template.xlsx` 模板文件。

### 2. 训练模型

```python
from trade_learner import TradeLearner

learner = TradeLearner()
training_data = learner.prepare_training_data('trade_history.xlsx')
learner.train_model(training_data)
```

### 3. 使用模型预测

```python
import akshare as ak

# 获取实时行情
current_data = ak.stock_zh_a_hist_min_em(symbol='000001', period='1')

# 预测买卖倾向
prediction = learner.predict(current_data)
print(f"建议操作: {prediction['action']}, 置信度: {prediction['confidence']:.2%}")
```

## 技术特征

模型使用以下技术指标作为特征：
- MA5/MA10/MA20 - 短中期趋势
- RSI - 超买超卖信号
- 成交量比率 - 资金活跃度
- 价格变化率 - 动量指标

## 建议

- 至少准备 50+ 条历史交易记录以获得较好效果
- 定期用新的交易记录重新训练模型
- 结合置信度和其他因素综合决策
- 可扩展更多技术指标提升准确度
