# 能源电力与AI基础设施

## 一句话解释

AI 算力不是虚拟的，它需要真实电力、土地、冷却、水资源、变电设备和数据中心基础设施。电力是智能系统落地的底层物理约束。

## 核心链条

模型需求 -> GPU集群 -> 数据中心 -> 电力容量 -> 散热系统 -> 电网接入 -> 资本开支

## 关键变量

| 变量 | 问题 |
| --- | --- |
| 功耗 | 单机柜、单集群耗电多少 |
| PUE | 数据中心总能耗效率 |
| 电网接入 | 当地电力容量和审批 |
| 电价 | 推理和训练成本 |
| 稳定性 | 是否能满足高可靠运行 |
| 低碳要求 | 是否需要绿电和储能 |

## 我的启发

当 AI 模型规模和推理量增长时，机会不只在芯片，也会扩散到电源、变压器、HVDC、液冷、储能、绿电和数据中心工程。
