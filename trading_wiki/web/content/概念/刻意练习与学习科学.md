# 刻意练习与学习科学

## 一句话解释

刻意练习是有目标、有反馈、针对弱点、持续迭代的训练。它不是机械重复，也不是只靠热情投入。

## 核心原理

| 要素 | 说明 |
| --- | --- |
| 明确目标 | 知道这次练什么 |
| 适度难度 | 略高于当前能力 |
| 即时反馈 | 快速知道哪里错 |
| 修正动作 | 针对错误调整 |
| 长期重复 | 让新模式稳定 |

## 学习闭环

输入 -> 理解 -> 输出 -> 反馈 -> 修正 -> 迁移

只有输入没有输出，知识停留在熟悉感；只有输出没有反馈，容易重复错误；只有反馈没有修正，复盘只是情绪安慰。

## 典型案例

- 学投资：不是看更多研报，而是写假设、跟踪证据、复盘失效条件。
- 学设计：不是收藏好图，而是拆解构图、色彩、层级并模仿重做。
- 学 AI：不是刷工具，而是用同一任务比较提示词、模型和工作流。

## 我的启发

个人知识库本质上是练习系统。每篇笔记都应该回答：我能用它解释什么、判断什么、创造什么、避免什么错误。
