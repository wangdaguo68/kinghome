# 哲学大师经典思想地图

> [!note] 使用边界
> 本页是个人知识库的入门地图，不替代哲学史教材。每位思想家的表述都只保留最核心的思想入口，后续可以继续拆成单独人物页、经典文本页和现实应用页。

## 一句话解释

哲学大师的价值不是提供标准答案，而是提供一套看世界的方法。学习哲学人物时，重点不是背名言，而是抓住：他在反对什么问题、提出什么核心概念、怎样改变了后来的思想结构。

## 中国哲学主线

| 人物 | 核心思想 | 经典关键词 | 现实启发 |
| --- | --- | --- | --- |
| 孔子 | 以仁为核心，以礼修身齐家治国，强调君子人格和关系秩序 | 仁、礼、君子、中庸、修身 | 做人做事要有稳定品格，关系和秩序本身也是能力 |
| 孟子 | 性善论，强调仁义、恻隐之心和民本政治 | 性善、四端、仁义、民贵君轻 | 人的道德感可以培养，组织和社会要重视人的尊严 |
| 老子 | 道法自然，反对过度人为控制，强调无为而治和柔弱胜刚强 | 道、无为、自然、反者道之动 | 很多系统不能靠强推，要顺势、留白、降低干预 |
| 庄子 | 逍遥游，强调齐物、心斋、坐忘，超越执着和固定分别 | 逍遥、齐物、无待、物化 | 不被单一身份和评价系统困住，获得更大的精神自由 |
| 王阳明 | 心即理，知行合一，致良知，强调内在道德觉察与行动统一 | 心学、知行合一、致良知 | 真正的认知必须落实为行动，知道但做不到不算真知 |

## 西方哲学主线

| 人物 | 核心思想 | 经典关键词 | 现实启发 |
| --- | --- | --- | --- |
| 苏格拉底 | 通过追问暴露无知，强调未经省察的人生不值得过 | 反诘法、德性、认识你自己 | 好问题比快答案重要，先看清自己不知道什么 |
| 柏拉图 | 理型论，认为感官世界背后有更真实的理念秩序 | 理型、洞穴寓言、灵魂、正义 | 现象背后有结构，教育是从影子走向真实 |
| 亚里士多德 | 从经验和分类出发，建立逻辑、目的论和德性伦理 | 逻辑、四因说、德性、中道 | 好生活来自长期实践形成的稳定品质 |
| 笛卡尔 | 方法怀疑，以“我思故我在”寻找确定性基础 | 怀疑、我思、理性主义、身心二元 | 面对复杂问题先清理不可靠前提 |
| 休谟 | 经验主义和怀疑论，强调因果只是习惯性联想，理性受情感驱动 | 经验、因果、怀疑、情感 | 很多所谓确定判断来自习惯和心理倾向 |
| 康德 | 调和理性主义与经验主义，认为人以先天范畴组织经验 | 先验、范畴、物自身、义务伦理 | 人不是被动接收世界，而是在结构化经验 |
| 黑格尔 | 辩证法和历史精神，认为矛盾推动意识和社会发展 | 辩证法、精神、否定之否定、历史 | 看问题要看矛盾、过程和历史展开 |
| 尼采 | 重估一切价值，批判奴隶道德，强调权力意志和超人 | 价值重估、权力意志、超人、永恒轮回 | 不盲从既有价值，要追问价值是谁创造的 |
| 马克思 | 历史唯物主义，资本批判，强调生产关系和阶级结构 | 异化、资本、阶级、生产关系 | 观念背后常有物质结构、利益关系和制度安排 |
| 海德格尔 | 追问存在本身，强调此在、向死而生和技术时代的遮蔽 | 存在、此在、向死而生、技术 | 人不只是工具性存在，要重新理解生活和存在方式 |
| 维特根斯坦 | 早期关注语言图像论，后期强调语言游戏和生活形式 | 语言游戏、生活形式、不可说 | 很多哲学困惑来自语言误用 |
| 福柯 | 分析权力、知识和规训机制，研究现代主体如何被塑造 | 权力、知识、规训、话语 | 看似自然的分类和制度背后有权力结构 |
| 弗洛伊德 | 精神分析，提出无意识、压抑、本我自我超我 | 无意识、压抑、梦、欲望 | 人并不完全透明，行为背后常有被压抑的动机 |
| 荣格 | 分析心理学，提出集体无意识、原型和个体化 | 原型、集体无意识、阴影、个体化 | 神话、梦和象征可以帮助理解人格深层结构 |

## 思想谱系怎么读

| 线索 | 代表人物 | 追问 |
| --- | --- | --- |
| 德性与修身 | 孔子、孟子、亚里士多德、王阳明 | 人如何成为更好的人 |
| 自然与自由 | 老子、庄子、尼采、海德格尔 | 人如何不被外在秩序完全吞没 |
| 理性与知识 | 苏格拉底、柏拉图、笛卡尔、康德 | 什么是真正可靠的知识 |
| 经验与怀疑 | 亚里士多德、休谟、维特根斯坦 | 我们如何从经验和语言中理解世界 |
| 历史与权力 | 黑格尔、马克思、福柯 | 人如何被时代、制度和话语塑造 |
| 无意识与象征 | 弗洛伊德、荣格、庄子 | 人如何理解自己看不见的内在动力 |

## 对个人知识库的用法

1. 搜一个哲学家时，先定位他回答的核心问题。
2. 再看他反对谁、继承谁、影响谁。
3. 把思想压缩成一个可迁移模型。
4. 写清它能用于生活、审美、品牌、交易或组织分析的哪一类问题。
5. 每个哲学家后续都可以拆成单页：思想定义、经典文本、相近概念、误读、现实用法。

## 常见误区

| 误区 | 更准确的处理 |
| --- | --- |
| 把哲学家当名言库 | 要理解问题、概念和论证结构 |
| 把中国哲学等同于人生鸡汤 | 中国哲学有严肃的伦理、政治、形上和修身结构 |
| 把西方哲学等同于抽象概念 | 西方哲学也在处理真实生活、社会和知识问题 |
| 用一个标签概括一个人 | 很多思想家内部有阶段变化和复杂争议 |
| 只学观点不学反对意见 | 哲学进步来自问题、反驳和重新定义 |
