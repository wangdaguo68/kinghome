# 心流与注意质量

## 一句话解释

心流是一种高度投入、反馈清晰、挑战与能力匹配的状态。它不是放松，也不是上瘾，而是注意力和行动高度协调。

## 心流条件

| 条件 | 说明 |
| --- | --- |
| 目标清楚 | 知道当前要完成什么 |
| 反馈及时 | 能看到进展或错误 |
| 挑战适中 | 太简单无聊，太难焦虑 |
| 干扰少 | 注意力不被频繁切走 |
| 行动连续 | 不断进入下一步 |

## 心流 vs 沉迷

| 项目 | 心流 | 沉迷 |
| --- | --- | --- |
| 结果 | 能力提升或作品完成 | 时间消耗和后悔 |
| 反馈 | 有建设性 | 多为即时刺激 |
| 控制感 | 更清醒 | 更被动 |
| 结束后 | 充实 | 空虚或疲惫 |

## 可迁移场景

- 学习：把任务切成可反馈的小块。
- 写作：先定问题和结构，再连续输出。
- 设计：减少工具切换，围绕一个视觉问题迭代。
- 交易研究：固定研究流程，避免盘中情绪刺激。
