# 习惯回路与行为改变

## 一句话解释

习惯回路是“线索 -> 行为 -> 奖赏”的自动化链条。改变习惯的关键不是单纯靠意志力，而是改线索、改环境、改反馈。

## 核心模型

| 环节 | 问题 | 例子 |
| --- | --- | --- |
| 线索 | 什么触发行为 | 无聊、压力、手机提醒 |
| 行为 | 自动做了什么 | 刷短视频、追涨、吃零食 |
| 奖赏 | 获得了什么 | 放松、刺激、逃避、确认感 |
| 身份 | 我把自己看成谁 | 自律的人、交易者、学习者 |

## 改变策略

- 减少坏习惯线索：把手机放远、关闭推送、降低交易频率。
- 增加好习惯线索：固定时间、固定地点、提前打开文档。
- 替换行为而不是只压制：焦虑时先记录而不是下单。
- 让反馈变短：学习、运动和复盘都需要可见进度。

## 我的启发

如果一个行为反复失败，先不要骂自己。更应该问：环境是否在持续奖励旧行为？有没有把新行为设计得足够容易启动？
