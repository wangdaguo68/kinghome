# 主线补涨退潮与风险收益比

## 一句话解释

主线是能持续吸引资金、产业逻辑和市场叙事共同支持的方向；补涨是主线扩散后的低位或滞涨环节；退潮是高位亏钱效应开始压制风险偏好。

## 判断主线

| 标准 | 问题 |
| --- | --- |
| 产业逻辑 | 是否有真实需求、订单、价格或政策 |
| 资金强度 | 龙头是否持续创新高 |
| 扩散结构 | 是否从核心环节扩散到材料、设备、应用 |
| 容量 | 是否容纳大资金 |
| 反复性 | 分歧后是否能修复 |

## 风险收益比

风险收益比不是“我觉得能涨多少”，而是：

- 上行空间是否大于下行风险。
- 失败时能否快速退出。
- 位置是否已经透支预期。
- 是否有反证和时间限制。

## 我的启发

主线不是买入理由，只是研究入口。买点必须来自赔率、证据和风险控制。越到补涨后期，越要降低权限。
