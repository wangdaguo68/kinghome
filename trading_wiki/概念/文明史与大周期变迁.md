# 文明史与大周期变迁

## 概念定义

文明史研究人类如何通过农业、城市、文字、国家、宗教、贸易、科学、工业、能源和信息技术组织大规模社会。大周期视角关注技术、资源、制度、人口和思想如何长期互动。

## 文明变迁的关键变量

| 变量 | 作用 |
| --- | --- |
| 能源 | 决定可动员的物理能力 |
| 技术 | 改变生产、战争、交通和知识 |
| 制度 | 决定协作、激励和权力分配 |
| 人口 | 影响劳动力、市场和社会压力 |
| 贸易 | 扩大分工和资源流动 |
| 知识 | 改变解释世界和控制世界的能力 |
| 信仰与意义 | 提供秩序、合法性和共同体 |

## 长周期观察框架

| 周期 | 观察问题 |
| --- | --- |
| 农业革命 | 定居、剩余、阶层和国家如何出现 |
| 轴心时代 | 宗教、哲学和伦理系统如何形成 |
| 商业扩张 | 贸易、金融和城市如何改变权力 |
| 科学革命 | 自然知识如何变成可积累体系 |
| 工业革命 | 能源和机器如何重构生产 |
| 信息革命 | 计算、网络和数据如何重构组织 |
| AI 时代 | 智能自动化如何改变知识和劳动 |

## 对未来判断的价值

文明史不是为了背年代，而是训练尺度感：

| 尺度感 | 用途 |
| --- | --- |
| 技术需要制度配套 | 避免只看发明不看采用 |
| 能源和材料是底层约束 | 避免只看软件叙事 |
| 思想会重塑组织 | 宗教、科学、民族、市场、AI 都会改变协作 |
| 周期中有路径依赖 | 旧系统不会轻易消失 |
| 大变迁常伴随冲突 | 利益重分配会带来政治和文化摩擦 |
