# 情绪周期与赚钱效应

## 一句话解释

情绪周期是市场参与者风险偏好在兴奋、扩散、分歧、退潮和修复之间循环。赚钱效应会吸引资金继续参与，亏钱效应会让资金收缩。

## 周期阶段

| 阶段 | 表现 |
| --- | --- |
| 启动 | 少数方向开始走强 |
| 扩散 | 赚钱效应吸引资金，题材蔓延 |
| 高潮 | 一致看多，后排补涨 |
| 分歧 | 龙头震荡，资金切换 |
| 退潮 | 高位亏钱效应扩散 |
| 修复 | 风险偏好重新恢复 |

## 我的启发

同一个产业逻辑，在不同情绪周期里权限不同。强逻辑也可能在退潮期杀估值，弱逻辑也可能在高潮期被资金推高。交易系统必须识别环境。
