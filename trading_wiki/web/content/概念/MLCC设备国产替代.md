# MLCC设备国产替代

## 概念定义

MLCC设备国产替代，不是单纯“MLCC涨价”，而是高端 MLCC 扩产中 **材料、工艺、设备、良率、客户认证** 全链条国产化的交易认知。

MLCC 是多层片式陶瓷电容器，核心工艺是把陶瓷介质层和内电极层交替堆叠、烧结、端接、测试。AI 服务器、AIDC 电源、汽车电子、5G/6G 通信提高高容、高压、高可靠、小型化 MLCC 需求后，国内产业链的瓶颈不只在成品产能，还在：

- 上游材料：钛酸钡/陶瓷粉体、镍粉/铜粉、电极浆料、离型膜、PET 基膜、外电极材料。
- 中游制造：三环集团、风华高科、火炬电子、宏达电子等成品厂的高端 MLCC 能力。
- 设备工艺：流延、印刷、叠层、层压/等静压、切割、排胶、烧结、测试、老化、分选。
- 客户认证：AI服务器、车规、通信设备等高可靠场景的认证周期和稳定交付。

当前最有价值的不是泛泛说“国产替代”，而是逐项验证：谁卡在材料，谁卡在设备，谁有真实高端客户，谁能把订单转成收入和毛利。

## 验证清单

- [ ] MLCC 需求来自 AI服务器、汽车电子、通信，还是普通消费电子复苏。
- [ ] 具体规格是高容、高压、车规、低压大容量、还是普通 MLCC。
- [ ] 国瓷材料粉体是否进入高端客户，外销比例和毛利率是否提升。
- [ ] 三环集团/风华高科高端 MLCC 产品是否量产、认证、放量。
- [ ] 设备订单是否真实存在，金额、客户、交付、验收节点是什么。
- [ ] 上游材料、设备、成品厂之间是否重复计算同一需求。
- [ ] 价格上涨能否穿透到公司合同价，而不是现货市场短期波动。

## 结论

MLCC设备国产替代不能只看设备，也不能只看涨价。最完整的交易链条应是：

```text
AI/车规/通信需求 -> 高端MLCC规格升级 -> 陶瓷粉体/镍粉/离型膜/浆料瓶颈 -> 成品厂扩产与良率 -> 国产设备导入 -> 订单/价格/毛利兑现
```

当前最值得沉淀的节点是：[[国瓷材料]] 的陶瓷粉体，[[三环集团]]/[[风华高科]] 的高端 MLCC 成品，[[洁美科技]]/[[博迁新材]] 的耗材材料，[[博杰股份]] 等设备验证。没有订单和收入确认前，全部保持观察或待验证。
