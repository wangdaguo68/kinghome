# 系统科学与复杂系统

## 概念定义

系统科学研究由多个要素、关系、反馈和边界组成的整体如何运作。复杂系统的行为常常不是部件简单相加，而是由互动、反馈、延迟和适应产生。

## 系统分析四件事

| 要素 | 问题 | 例子 |
| --- | --- | --- |
| 元素 | 系统由什么组成 | 公司、用户、供应商、监管 |
| 关系 | 元素如何连接 | 资金流、信息流、物流 |
| 反馈 | 行动如何反过来影响系统 | 正反馈扩张、负反馈稳定 |
| 边界 | 你把什么算进系统 | 行业、生态、社会、自然环境 |

## 复杂系统特征

| 特征 | 说明 |
| --- | --- |
| 非线性 | 小变化可能产生大后果，大投入也可能无效 |
| 延迟 | 行动和结果之间存在时间差 |
| 涌现 | 整体行为不是单个部件能解释 |
| 适应 | 参与者会学习和调整策略 |
| 路径依赖 | 历史选择会限制未来 |

## 杠杆点

| 杠杆层级 | 例子 |
| --- | --- |
| 参数 | 价格、补贴、利率 |
| 流量 | 资金、人才、数据、产能 |
| 反馈回路 | 推荐算法、复购机制、网络效应 |
| 规则 | 标准、监管、合同、激励 |
| 目标 | 公司或系统追求什么 |
| 范式 | 系统默认相信什么 |

## 与产业研究的连接

产业链不是静态上下游表，而是一个复杂系统。价格、库存、产能、政策、客户认证、技术路径和资本开支都有延迟和反馈。详见[[AI全产业链地图]]、[[半导体设备国产替代全景]]。
