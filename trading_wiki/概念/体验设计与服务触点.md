# 体验设计与服务触点

## 概念定义

体验设计关注人在使用产品、服务或空间时的完整感受链条。它不仅包括界面，还包括预期、进入、理解、行动、反馈、等待、错误、售后和记忆。

## 体验的时间链条

| 阶段 | 用户问题 | 设计重点 |
| --- | --- | --- |
| 进入前 | 我为什么要来？ | 价值主张、品牌承诺 |
| 第一眼 | 这是什么？可信么？ | 信息层级、视觉信任 |
| 开始使用 | 我该怎么做？ | 导航、引导、默认选项 |
| 执行动作 | 是否顺畅？ | 交互、反馈、速度 |
| 出错 | 我能恢复吗？ | 错误提示、撤销、客服 |
| 完成后 | 我记住了什么？ | 结果反馈、情绪峰终、复访 |

## 服务触点

| 触点 | 例子 | 风险 |
| --- | --- | --- |
| 数字触点 | App、官网、小程序、邮件 | 信息混乱、加载慢 |
| 人员触点 | 客服、销售、顾问 | 话术不一致、信任破裂 |
| 空间触点 | 门店、展厅、活动 | 氛围和品牌不匹配 |
| 物理触点 | 包装、票据、设备 | 材质廉价、使用不顺 |
| 规则触点 | 退款、保修、会员 | 承诺和实际不一致 |

## 好体验的判断

| 标准 | 说明 |
| --- | --- |
| 可发现 | 用户能找到入口 |
| 可理解 | 用户知道它是什么 |
| 可行动 | 用户知道下一步 |
| 可恢复 | 出错后能补救 |
| 可信任 | 承诺、反馈和结果一致 |
| 可记忆 | 体验结束后留下明确印象 |

## 与品牌的关系

品牌承诺如果不能在体验触点兑现，就会变成空话。体验是品牌的证据，品牌是体验的长期记忆。
