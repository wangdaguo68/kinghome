# 佛教基础与释迦牟尼

## 概念定义

佛教起源于古印度，传统上以释迦牟尼佛的觉悟和教导为源头。佛学是对佛教教义、经典、修行经验、历史宗派和哲学问题的系统研究。

| 名词 | 含义 |
| --- | --- |
| 佛 | 觉悟者，不是造物主 |
| 法 | 佛陀所发现和教导的规律、路径与实践 |
| 僧 | 修行共同体，最初指僧团，广义可指依教修学者 |
| 三宝 | 佛、法、僧 |
| 三藏 | 经、律、论 |

## 释迦牟尼关注的问题

释迦牟尼不是从宇宙起源开始建立体系，而是从人的现实经验开始：生老病死、爱别离、怨憎会、求不得、五取蕴炽盛。

| 观察对象 | 佛学问题 | 对应笔记 |
| --- | --- | --- |
| 痛苦 | 痛苦如何出现、如何被止息 | [[四圣谛与八正道]] |
| 欲望 | 为什么越抓越苦 | [[烦恼三毒与苦的生成]] |
| 自我 | “我”是否固定不变 | [[三法印与无我]] |
| 经验 | 身心经验由哪些部分组成 | [[五蕴六处十八界]] |
| 行为 | 行为如何塑造后续经验 | [[业力轮回与解脱]] |

## 佛教不是单一系统

| 传统 | 重点 | 常见代表 |
| --- | --- | --- |
| 早期佛教 | 四圣谛、八正道、缘起、戒定慧 | 阿含、尼柯耶 |
| 南传上座部 | 巴利经典、内观、阿毗达摩 | 斯里兰卡、泰国、缅甸传统 |
| 大乘佛教 | 空性、菩萨道、慈悲与智慧 | 中观、瑜伽行、净土、禅、华严 |
| 金刚乘 | 密续、本尊、仪轨、上师传承 | 藏传佛教部分传统 |
| 汉传佛教 | 经典翻译、禅净教律融合 | 禅宗、净土、天台、华严 |

学习佛学时要问：这句话来自哪个传统？是经典原文、祖师解释、现代讲法，还是个人心得？

## 初学最容易混淆的点

| 混淆 | 更准确的说法 |
| --- | --- |
| 佛是神 | 佛更接近觉悟者和老师，不是造物主 |
| 佛教只讲来世 | 早期教义非常重视当下经验中的苦和止苦 |
| 佛学就是哲学 | 佛学包含哲学，但核心是可实践的修行道路 |
| 佛学就是心理学 | 二者可以对话，但佛学有解脱论和伦理实践维度 |
| 信佛等于懂佛学 | 信仰、理解、实践、证悟是不同层次 |

## 学习方法

1. 先建立[[佛学全景地图]]。
2. 再读[[四圣谛与八正道]]，不要一开始钻宗派名词。
3. 用[[止观禅修与正念]]做生活观察。
4. 遇到“空性、无我、业力、涅槃”这类词，不要按字面理解，进入对应笔记逐层拆解。
