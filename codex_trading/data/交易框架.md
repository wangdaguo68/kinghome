# 交易框架

> 核心理念：**价值选股，情绪交易。用客观规则破我执。**
>
> 体系闭环：**880005红盘家数 → MA5/MA3周期标签 → 匹配交易模式 → 对应仓位 → 四阶段时间窗口执行 → 量能盘口验证 → 风控兜底**

---

## 一、情绪周期定位系统

### 1.1 核心指标

| 指标 | 参数 | 量化标准 | 作用 |
|------|------|----------|------|
| **880005红盘家数（RedCount）** | 实时数据 | 冰点：≤1000家；高潮：≥4000家 | 触发日内/日线极值信号，启动对应模式 |
| **880005 MA5** | 周期5 | 下轨/冰点区：MA5 < 2000家 → 高胜率进攻节点；上轨/高潮区：MA5 > 2300-2500家 → 风险警示区 | 划分情绪阶段，决定操作节奏和仓位 |
| **880005 MA3** | 周期3 | 与MA5结合判断拐点，辅助确认短期情绪波动 | 细化日内/隔日节奏，拐点确认辅助信号 |
| **同花顺全A股CCI(5)** | 周期5 | 买入信号：从-100附近向上拐头；卖出信号：到+100以上或从上轨向下拐头 | 量化市辅助判断轮动节奏，反应更灵敏 |
| **昨日炸板股指数** | 自身5日线 | 可上仓位：指数远离5日线向上；必须降仓：指数接近/跌破5日线 | 辅助判断市场短线修复力和容错率 |

### 1.2 八大周期标签

| 周期标签 | 定量标准 | 市场意义 | 操作基调 |
|----------|----------|----------|----------|
| **冰点 IcePoint** | RedCount ≤ 1000，MA5 < 2000 | 全市场普跌恐慌，盈亏比极高 | 进攻：寻找买点，开仓/加仓 |
| **拐点向上 TurnUp** | MA5从低位拐头，当日RedCount > 5天前 | 新周期起点确认信号 | 重仓出击；MA3共振=最高胜率节点 |
| **主升浪 MainRally** | MA5明确上升，数值持续攀升 | 情绪与赚钱效应持续扩散的黄金做多期 | 聚焦核心容量品种，积极持股 |
| **高潮 Climax** | RedCount ≥ 4000，MA5 > 2300-2500 | 全市场亢奋，盈亏比急剧恶化 | 坚决防守，禁高位接力 |
| **拐点向下 TurnDown** | MA5从高位拐头，当日RedCount < 5天前 | 退潮期开始确认 | 系统降仓、控风险 |
| **退潮 Downtrend** | MA5明确下降，数值持续回落 | 亏钱效应扩散，操作难度大 | 严控仓位，仅极小仓位博弈 |
| **低位震荡 LowShake** | MA5在1800-2000区间横向波动 | 情绪低迷但未创新低，磨底状态 | 积极选股，不急于重仓 |
| **高位震荡 HighShake** | MA5在2500-2800区间横向波动 | 情绪高位滞涨，风险积聚期 | 分仓、去弱留强、降单票仓位 |

### 1.3 MA3与MA5组合判断

| MA5趋势 | MA3趋势 | 周期状态 | 操作策略 |
|----------|----------|----------|----------|
| 向上 | 向上 | **主升共振** | 最强主升浪，重仓聚焦主线龙头 |
| 向上 | 向下 | **下跌中继/反弹** | 整体退潮中短期修复，只做套利，绝不格局，快进快出 |
| 向下 | 向上 | **上升中继/分歧** | 整体上升中短期退潮，去弱留强，关注核心票低吸 |
| 向下 | 向下 | **双杀退潮** | 最强退潮期，坚决防守，空仓或极小仓位，等待冰点 |

### 1.4 MA5拐点预判方法

- 若次日红盘家数 > 5天前（即将被剔除出均值的第6天）的数值，则MA5将向上拐头
- 若次日红盘家数 < 5天前的数值，则MA5将向下拐头
- **最佳信号**：5日线与3日线同时从低位拐头向上 = 短中期情绪共振回暖，胜率极高

### 1.5 日线级别典型三日循环

| 日期 | 特征 | 操作 |
|------|------|------|
| **Day1 极值日** | 尾盘RedCount ≤ 1000，情绪冰点 | 尾盘制定次日反弹计划，或盘中执行"分歧三板低吸" |
| **Day2 反弹日** | 情绪从冰点修复，高位股亏钱效应停止扩散 | 执行反包首板、最高板弱转强、大长腿低吸 |
| **Day3 强化日** | 情绪进一步演绎 | 若仍在相对低位：做最快一进二；若已到高热区域：放弃开仓，进入防守 |

## 二、六大交易模式

> **核心原则：每种模式只在对应的周期里启用，其余时间禁用。模式外操作是亏损的首要根源。**

### 2.1 首板打板（FirstLimit）

- **核心逻辑**：赚取市场对逻辑的二次扩散溢价，避免潜伏的不确定性
- **启用周期**：拐点向上、主升浪初期、低位震荡独立走强日
- **禁用周期**：高潮、拐点向下、退潮
- **选股分类**：
  - **共振首板**：与880005/指数分时低点同步启动涨停，选板块内涨幅第一、逻辑最正的标的
  - **唯一性逻辑首板**：新题材/新事件驱动，必须是该逻辑最直接、唯一受益标的
- **量能标准**：
  - 成交额 ≥ 10亿元（流动性安全底线），优先20亿附近
  - 竞价成交额 > 2000万、量比 > 1.5为活跃线
- **盘口买点**：
  - 缩量板：排队策略
  - 放量板（炸板回封）：首次封板无量则放弃，等待**二次回封扫板**
- **封板确认**：容量首板早盘封板，板上有3-4亿封单视为强度确认；超过1亿主动性扫单是合力确认信号
- **仓位**：小票（2-4亿成交）轻仓5-10%，大票可适当重仓

### 2.2 一进二（1To2）

- **核心逻辑**：捕捉新龙头确认日
- **启用周期**：拐点向上确认后强化日、主升浪初期题材发酵日
- **禁用周期**：高潮后、退潮
- **买点**：最快换手二板扫板
- **竞价信号**：竞价成交量达前一日总成交量15%以上，且高开5%以上（小容量，首板成交额≤8000万）；2022年后优先选成交额≥10亿品种

### 2.3 二波反包（SecondWaveWrap）

- **核心逻辑**：冰点后博弈前期绝对人气龙头/核心容量品种修复弹性
- **启用周期**：冰点次日、低位震荡末期
- **禁用周期**：高位震荡、高潮、退潮初期
- **标的筛选**：
  - 老龙头反包：前期有绝对人气和辨识度的连板龙头
  - 核心容量反包：上一轮行情中成交额持续领先的板块中军
- **触发条件**：前一交易日RedCount ≤ 1000
- **买点**：打最早上板的反包首板；首次上板缩量则等二封回封
- **注意**：反包首板若缩量过快 → 筹码交换不充分，次日抛压大，需谨慎

### 2.4 分歧低吸（DipBuy）

- **核心逻辑**：龙头恐慌承接
- **启用周期**：主升浪首次强分歧、拐点向上初期核心票温和换手日
- **禁用周期**：退潮任何低吸、下午指数预期回落时
- **买点**：破均线承接、情绪博弈低吸、日内衰竭点低吸
- **核心禁忌**：**退潮期绝不低吸，下午指数回落时绝不低吸**

### 2.5 极值套利（ExtremeArbitrage）

- **核心逻辑**：利用情绪"钟摆"两端的过度反应，反向操作
- **启用周期**：冰点做多套利，高潮做空/防守套利
- **禁用周期**：情绪处于中间模糊地带（震荡期）
- **冰点套利**：
  - 阈值：RedCount ≤ 1000
  - 次日重点做"反包首板"或"最高板的弱转强"
- **高潮套利**：
  - 阈值：RedCount ≥ 4000
  - 当日或次日放弃所有中高位接力，仅可轻仓试错低位首板
- **极值日操作**：
  - 极值日：聚焦最强方向，博弈"分歧三板"烂板低吸（"3+1"模式）
  - 反弹日：主攻首板或反包板；前日烂板顽强的"二板"可博弈弱转强

### 2.6 拿货分时（AccumulationIntraday）

- **核心逻辑**：识别主力在情绪分歧点对核心品种的主动承接与筹码收集
- **适用范围**：仅限连续一字板后开板的个股
- **启用周期**：高潮转退潮的指数大跌日、高位震荡中
- **禁用周期**：主升浪、单边退潮
- **识别特征**：
  - 当日指数单边下跌，但该股逆势震荡走高，多次触及涨停却故意不封死
  - 日内振幅通常 > 10%，完成充分换手
  - 日K线呈放量阳线 = 主力主动承接筹码信号
- **操作节奏**：当日分时可轻仓试错；次日若缩量加速，竞价低开是直接加仓点

## 三、四时间窗口操作规则

| 窗口 | 时间 | 核心任务 | 操作要点 | 禁忌 |
|------|------|----------|---------|------|
| **竞价窗口** | 9:15-9:30 | 判断开盘极值，规划全天基调 | 早盘冰点(RedCount<1000)：做最高最强，优先级：最高板分歧低吸 > 竞价最高二板 > 有连板预期首板 > 烂板三板分歧低吸；高潮顶值(RedCount>3000-4000)：禁高位接力 | 不在竞价做买卖决策 |
| **开盘窗口** | 9:30-10:30 | 验证竞价判断，执行计划 | 共振日确认：成交额榜前5集体走强+指数共振→可执行核心龙头缩量加速板；冰点次日：执行最早上板的反包首板、最高板弱转强 | 不追涨停板（除非计划内首板打板） |
| **盘中窗口** | 10:30-14:30 | 处理轮动、回避风险 | 存量环境下低吸前一日最弱板块核心标的；突发题材参与最快上板的容量首板 | **严禁下午指数回落时做任何低吸/弱转强** |
| **尾盘窗口** | 14:30-15:00 | 确认全天情绪，制定次日计划 | 极值确认(RedCount<1000)：①人气龙头尾盘炸板回落至零轴附近低吸；②满足封单标准的跌停翘板；锁定次日计划 | 不在尾盘冲动买入非计划标的 |

## 四、仓位管理

### 4.1 基于情绪周期的仓位上限

| CycleTag | 总仓位 | 单票上限 | 说明 |
|----------|--------|---------|------|
| 冰点反转 | 可满仓 | 核心龙头可重仓 | MA5低位拐头，RedCount≤1000 |
| 主升浪 | 积极持股 | 核心票可重仓 | MA5明确上升期 |
| 高潮 | 减仓至2-3成 | 仅极小仓位试错 | RedCount≥4000，MA5到上轨 |
| 退潮 | 空仓或≤1成 | 禁止中高位接力 | MA5明确下降，双杀时空仓 |
| 震荡/存量 | 10%-20% | 单票1-2成 | 严禁追高 |
| 连续亏损3笔 | 强制半仓 | 半仓操作 | 直到胜率回升 |
| 核吸操作 | 先1/3仓位 | 确认后再加仓 | 不一把梭哈 |

### 4.2 不同市场生态仓位配置

| 市场类型 | 特征 | 策略 |
|----------|------|------|
| **增量市场** | 成交量显著放大，有清晰主线 | 聚焦主线，共振点重仓，单票可推至半仓以上 |
| **存量/轮动市场** | 成交量低迷，热点快速轮动 | 控仓或分仓切换，总仓10%-20%，单票1-2成，严禁追高 |

### 4.3 账户级风控

| 风控项 | 阈值 | 动作 |
|--------|------|------|
| 单笔硬止损 | -5% | 到就走，不犹豫 |
| 连续回撤 | 5%-7% | 自动降一档仓位 |
| 连续亏损3笔 | — | 强制半仓操作 |
| 最大回撤铁律 | 12% | 账户风控底线 |
| 割肉后 | — | 强制空仓1天 |
| 执行混乱时 | — | 降仓位做，满仓标准降为5成 |

## 五、买入标准化流程

每笔买入前，必须逐项确认：

```
□ 标的：XXX
□ 当前CycleTag：[冰点/拐点向上/主升浪/高潮/退潮/震荡]
□ 触发模式：[首板打板/一进二/二波反包/分歧低吸/极值套利/拿货分时]
□ 该模式在当前CycleTag下是否启用：[是 → 继续 / 否 → 禁止买入]
□ 容量标准：成交额___亿（≥10亿？___）
□ 竞价额：___万（>2000万？___）
□ 量比：___（>1.5？___）
□ 买入理由（一句话，非价格描述）：XXX
□ 计划仓位：___%
□ 止损位：-___%（到就走）
```

**全部确认通过方可下单。任何一项为"否"，禁止买入。**

## 六、卖出规则

| 卖出类型 | 触发条件 | 适用场景 |
|----------|---------|---------|
| 硬止损 | 浮亏达到-5% | 所有持仓，无条件执行 |
| 逻辑证伪止损 | 买入逻辑被破坏（情绪不升反降/板块强度滑落） | 无论盈亏果断离场 |
| 时间止损 | 买入后至次日午盘仍无溢价表现 | 模式内交易（首板、一进二），不格局 |
| 技术位止损 | 跌破关键分时均线或日线平台下沿，无力快速收回 | 容量票 |
| 周期拐点卖出 | CycleTag从主升浪/高潮转为拐点向下 | 无条件减仓高位持仓 |
| 情绪高潮卖出 | RedCount > 4000 | 禁开新仓，去弱留强，仅保留绝对核心 |
| 减仓锁利 | 涨至目标位减半仓 | 锁定部分利润，剩余仓持趋势 |
| 量化信号卖出 | 容量票大涨后次日成交量大幅萎缩 | 量化资金已离场 |

## 七、容量品种筛选标准

- **核心门槛**：日成交额 ≥ 10亿元，20亿左右为黄金区间
- **缩量/低迷环境**：提升至"成交额≥10亿 + 流通市值≥100亿"
- **优先选择**：板块内涨幅第一、逻辑最正的标的
- **"绕不开的品种"**：在选定赛道中选择市场共识度高、具备产业核心地位的标的
- **简单逻辑优先**：选择越简单、越能被广大投资者理解的行业和逻辑，弹性和传播性更好
- **弱者思维执行**：重点参与机构或游资重仓封板的容量票，视为确定性信号
- **大票锚定**：开盘观察"成交额排行榜"前20名走势，前排涨=增量，前排跌=存量

## 八、分时图与盘口验证

### 8.1 分时核心形态

| 形态 | 定义 | 买卖参考 |
|------|------|---------|
| 分时高点 | 第一次顶点为主高，回调后再次上攻顶点为次高 | 次高常为日内卖点，一波拉升通常只有两波高点 |
| 分时低点 | 首次杀跌谷底为主低，反弹后二次下探为次低 | 次低常为日内买点 |
| 均价线（生命线） | 日内强弱核心参考线 | 杀破后形成W底/双底拉回站稳 = 分歧低吸买点 |
| 拿货分时 | 指数大跌日个股逆势放量震荡，触板不封 | 主力主动承接信号，次日缩量加速为加仓点 |

### 8.2 量能盘口标准

| 场景 | 量化标准 |
|------|---------|
| 竞价量能 | 竞价成交额>2000万、量比>1.5为活跃线 |
| 盘中承接 | 单波上冲超2%，第一次回调幅度不超过涨幅1/3；均线处买盘挂单厚重；卖单被连续万手买单吃光 |
| 涨停板质量 | 容量票早盘封板封单3-4亿为强度确认；弱势环境封单超15亿需警惕；回封需5分钟内完成+大单扫货 |
| 翘板量能 | 跌停板附近累积成交量达历史天量日15%-25% |
| 竞价窗口细节 | 9:20前大单撤单多为虚假试探，9:20后挂单稳步增加且不撤=意图真实 |

## 九、戒律清单

> **以下规则无条件遵守，违反即为模式外操作，是亏损的首要根源。**

1. **连续亏损2笔后，不买**
2. **割肉当天，不买新票**
3. **MA5拐头向下（退潮期），不做任何低吸**
4. **下午指数回落时，不做任何买入**
5. **标的不在主线前三，不买**
6. **CycleTag下该模式未启用，不买**
7. **容量标准不达标（成交额<10亿），不买**
8. **没有设止损位的交易，不做**
9. **高潮期（RedCount≥4000），禁中高位接力**
10. **双杀退潮（MA3↓MA5↓），空仓等待冰点**

## 十、量化标准速查表

| 标准项 | 阈值 | 作用 |
|--------|------|------|
| 冰点进攻 | RedCount ≤ 1000，MA5 < 2000拐头向上 | 系统级买入信号 |
| 高潮防守 | RedCount ≥ 4000，MA5 > 2300-2500拐头向下 | 系统级卖出信号 |
| 容量门槛 | 成交额 ≥ 10亿（优选20亿） | 确保流动性 |
| 竞价筛选 | 竞价额 > 2000万；量比 > 1.5 | 筛选活跃品种 |
| 一进二竞价 | 竞价量 ≥ 前日15%+高开5% | 龙头确认信号 |
| 撬跌停公式 | 翘板量 ≈ 历史天量日成交15%-25% | 博弈地天板 |
| 拿货分时 | 逆势日，振幅>10%且触板不封 | 识别主力拿货 |
| 硬止损 | -5% | 单笔风控底线 |

| 回撤降档 | 连续回撤5%-7% | 自动降一档仓位 |
| 情绪容错率 | 昨日炸板股指数远离其MA5 | 量化修复力 |

## 十一、五大可复用原则

1. **节奏至上**：精准踏准情绪周期节奏，节奏优于个股选择，决定仓位轻重
2. **高胜率模式聚焦**：仅在高胜率验证过的模式上重仓，摒弃随意交易，靠概率复利取胜
3. **生态区分**：增量（格局持股）vs 存量（高频切换），采取对应策略
4. **仓位匹配**：高确定性机会推重仓，模糊/退潮期控仓
5. **持续进化**：随市场生态更新模式，无"一招鲜"

## 十二、模式进化路径

- **买点前置**：当模式被市场广泛模仿后，主动将买点提前（如从打板改为预判性低吸）
- **交割单反思**：定期分析亏损来源，优化参数调整，强制减少非模式内操作
- **资源整合**：利用核心圈获取深度信息，但决策前必须独立验证
