# A股超短线「能量交易」量化策略需求文档

> 目标：将“市场能量”这个主观交易框架，拆解成可量化、可回测、可执行的超短线策略模型。  
> 适用市场：A股。  
> 策略风格：超短线、事件驱动、情绪周期、题材轮动、龙头/前排优先。  
> 重要声明：本文档用于策略开发和研究，不构成投资建议。实盘前必须完成充分回测、模拟盘验证和风控压力测试。

---

## 1. 策略核心思想

A股超短线交易可以理解为一个“能量转移游戏”。

短期股价并不完全由估值决定，而主要由以下四类能量共同推动：

1. 资金能量：成交额、换手率、资金流入、封单强度。
2. 情绪能量：涨停数量、连板高度、炸板率、跌停数量、昨日涨停溢价。
3. 题材能量：板块涨停家数、板块成交额、板块扩散程度、核心龙头强度。
4. 筹码能量：位置高低、放量结构、承接强弱、断板反馈。

超短线策略的目标不是寻找长期低估资产，而是识别：

> 市场能量正在流向哪里，哪个方向最强，哪只股票是能量核心，以及能量是否即将衰竭。

策略最终要解决三个问题：

1. 今天市场是否适合做超短线？
2. 今天资金能量主要集中在哪个题材或板块？
3. 该题材中最值得参与的前排股票是谁？

## 2. 策略交易周期

### 2.1 持仓周期

- 默认持仓：1 到 3 个交易日。
- 极强龙头可适当延长，但需要满足持续超预期条件。
- 若次日明显低于预期，应快速止损或止盈。

### 2.2 交易频率

- 不是每天都交易。
- 只在市场短线能量评分达到阈值时开仓。
- 市场进入退潮期时空仓。

### 2.3 交易对象

优先考虑：

- 主板、创业板、科创板中的高流动性股票。
- 当日强势题材前排股。
- 涨停股、连板股、强趋势股、弱转强个股。

排除：

- ST股。
- 退市风险股。
- 长期成交额过低的股票。
- 一字涨停无法买入的股票。
- 极端高位且已经出现明显亏钱效应的股票。

## 3. 市场能量评分模型

市场能量用于判断今天是否适合超短线交易。

### 3.1 输入指标

每日收盘后统计以下数据：

| 指标 | 含义 | 方向 |
|---|---|---|
| limit_up_count | 涨停家数 | 越高越好 |
| limit_down_count | 跌停家数 | 越低越好 |
| failed_limit_up_rate | 炸板率 | 越低越好 |
| max_consecutive_boards | 最高连板高度 | 越高越好 |
| yesterday_limit_up_avg_return | 昨日涨停今日平均收益 | 越高越好 |
| high_board_next_day_return | 高位连板次日反馈 | 越高越好 |
| market_turnover | 全市场成交额 | 温和放大较好 |
| index_trend_score | 指数趋势状态 | 越强越好 |

### 3.2 市场能量评分公式

可先使用简单打分模型：

```text
MarketEnergyScore = 
    0.20 * 涨停家数评分
  + 0.15 * 连板高度评分
  + 0.20 * 昨日涨停溢价评分
  + 0.15 * 炸板率反向评分
  + 0.10 * 跌停家数反向评分
  + 0.10 * 市场成交额评分
  + 0.10 * 指数环境评分
```

评分范围：0 到 100。

### 3.3 市场状态划分

| MarketEnergyScore | 市场状态 | 策略动作 |
|---|---|---|
| 80 - 100 | 强能量期 | 可以积极参与主线前排 |
| 60 - 80 | 可交易期 | 控制仓位参与核心标的 |
| 40 - 60 | 混沌期 | 轻仓试错，优先低吸或不交易 |
| 0 - 40 | 退潮期 | 空仓或只做观察 |

## 4. 题材能量评分模型

题材能量用于判断资金正在流向哪个方向。

### 4.1 板块/题材候选池

每日根据以下方式构建题材池：

1. 行业板块。
2. 概念板块。
3. 涨停原因归类。
4. 新闻事件主题。
5. 自定义题材映射表。

### 4.2 输入指标

| 指标 | 含义 | 方向 |
|---|---|---|
| sector_limit_up_count | 板块涨停家数 | 越高越好 |
| sector_avg_return | 板块平均涨幅 | 越高越好 |
| sector_turnover_growth | 板块成交额增速 | 越高越好 |
| sector_top_stock_return | 板块最强股涨幅 | 越高越好 |
| sector_board_spread | 板块扩散程度 | 越高越好 |
| sector_continuity_days | 板块连续活跃天数 | 2-5天较好 |
| leader_strength_score | 龙头强度 | 越高越好 |
| media_heat_score | 消息/舆情热度 | 适度较好 |

### 4.3 题材能量评分公式

```text
SectorEnergyScore =
    0.25 * 板块涨停家数评分
  + 0.15 * 板块平均涨幅评分
  + 0.15 * 板块成交额增速评分
  + 0.20 * 龙头强度评分
  + 0.10 * 板块扩散评分
  + 0.10 * 持续性评分
  + 0.05 * 舆情热度评分
```

### 4.4 题材阶段识别

| 阶段 | 特征 | 策略动作 |
|---|---|---|
| 点火期 | 少数个股涨停，板块刚放量 | 小仓试错核心首板/低位前排 |
| 发酵期 | 龙头连板，板块扩散，赚钱效应增强 | 重点参与龙头或前排换手板 |
| 高潮期 | 板块批量涨停，后排乱飞 | 谨慎追高，只看龙头超预期机会 |
| 衰竭期 | 龙头断板，炸板率上升，后排大跌 | 停止接力，等待新主线 |

## 5. 个股能量评分模型

个股能量用于选择具体买入标的。

### 5.1 个股基础过滤条件

候选股必须满足：

```text
非ST
非退市风险
上市时间 > 60个交易日
近20日平均成交额 > 2亿元
当日成交额 > 3亿元
当日涨幅 > 5%
所属题材 SectorEnergyScore 排名前3
```

参数可根据回测结果调整。

### 5.2 个股能量指标

| 指标 | 含义 | 方向 |
|---|---|---|
| price_strength | 当日涨幅强度 | 越高越好 |
| volume_ratio | 量比 | 适度放大较好 |
| turnover_rate | 换手率 | 适度较好 |
| limit_up_quality | 涨停质量 | 越高越好 |
| intraday_resilience | 分时承接强度 | 越高越好 |
| rank_in_sector | 板块内强度排名 | 越靠前越好 |
| consecutive_boards | 连板数 | 根据周期判断 |
| gap_strength | 次日竞价强度 | 越强越好 |
| chip_position | 筹码位置 | 中低位更安全 |
| leader_identity | 是否龙头/前排 | 是更好 |

### 5.3 个股能量评分公式

```text
StockEnergyScore =
    0.20 * 个股涨幅强度评分
  + 0.15 * 成交量能评分
  + 0.15 * 涨停质量评分
  + 0.15 * 板块内排名评分
  + 0.15 * 分时承接评分
  + 0.10 * 龙头辨识度评分
  + 0.10 * 筹码位置评分
```

## 6. 龙头识别规则

龙头是市场能量的核心承载器。

### 6.1 龙头候选条件

满足以下条件越多，越可能是龙头：

1. 所属题材能量排名市场前3。
2. 个股启动时间早于同板块大多数股票。
3. 连板高度处于板块第一梯队。
4. 分歧时仍能回封或强势承接。
5. 断板后仍有资金反包尝试。
6. 板块涨跌明显受它影响。
7. 市场讨论度高。
8. 次日溢价明显强于后排。

### 6.2 龙头评分公式

```text
LeaderScore =
    0.25 * 连板高度排名评分
  + 0.20 * 启动时间领先评分
  + 0.20 * 涨停质量评分
  + 0.15 * 板块影响力评分
  + 0.10 * 次日溢价评分
  + 0.10 * 市场辨识度评分
```

### 6.3 龙头优先级

买入优先级：

```text
主线龙头 > 主线前排 > 主线低位补涨 > 非主线独立强势股 > 后排跟风股
```

原则：

> 宁愿错过后排杂毛，也不要在能量衰竭期追高买弱股。

## 7. 买入信号设计

### 7.1 模式一：主线发酵期前排接力

适用条件：

```text
MarketEnergyScore >= 60
SectorEnergyScore 排名前3
题材阶段 = 发酵期
StockEnergyScore >= 70
LeaderScore >= 60
昨日涨停今日溢价为正
炸板率不高于过去20日均值
```

买入方式：

1. 竞价高开 2% 到 7%，且成交额明显放大。
2. 开盘后快速上攻并突破昨日高点。
3. 分时回踩均价线不破，重新放量上攻。
4. 涨停回封时买入，或强势分时确认后买入。

避免买入：

- 高开超过 9% 且快速跳水。
- 板块只有单只股票强，没有助攻。
- 后排先涨，龙头走弱。
- 前一日板块已经严重高潮。

---

### 7.2 模式二：龙头分歧转一致

适用条件：

```text
个股为题材龙头
前一日出现分歧或烂板
次日没有大幅低开
开盘后承接强
板块没有明显退潮
市场短线能量仍处于可交易区间
```

买入方式：

1. 早盘低开不破关键支撑后快速翻红。
2. 分时出现放量拉升。
3. 板块内助攻股同步走强。
4. 个股重新封板或突破分时高点。

核心逻辑：

> 龙头该弱不弱，就是超预期。超预期会吸引短线资金回流。

---

### 7.3 模式三：主线低位补涨

适用条件：

```text
主线龙头已经打开高度
板块仍在发酵或高潮初期
低位个股出现首板或强势放量突破
个股位置低于板块核心高标
```

买入方式：

1. 选择同题材低位、低市值、高弹性股票。
2. 首板放量且封板质量高。
3. 次日竞价强于预期可考虑接力。

风险提示：

- 补涨通常生命周期短。
- 龙头断板后，补涨股容易快速退潮。
- 高潮末端的补涨最容易坑人。

---

### 7.4 模式四：新题材点火试错

适用条件：

```text
市场无明确主线
新消息刺激出现
相关个股首板增加
题材成交额快速放大
```

买入方式：

1. 小仓位参与最先涨停或最强换手首板。
2. 次日若题材扩散则加大关注。
3. 若无扩散，快速止盈或止损。

仓位：

- 点火期只允许小仓试错。
- 不允许重仓赌题材成为主线。

## 8. 卖出信号设计

### 8.1 个股卖出条件

出现以下任一条件，触发卖出或减仓：

```text
个股低于预期低开超过 -5%
开盘后无法翻红且板块走弱
跌破分时均价线后无法收回
炸板后无法回封
龙头断板且承接弱
放量滞涨或高位长上影
次日无溢价
```

### 8.2 板块卖出条件

```text
题材龙头断板
板块涨停家数大幅减少
后排个股大面积低开
板块成交额放大但指数不涨
题材从发酵期进入高潮末端或衰竭期
```

### 8.3 市场卖出条件

```text
MarketEnergyScore < 40
跌停家数明显增加
昨日涨停平均收益为负
高位连板股出现连续大面
炸板率显著升高
```

## 9. 仓位管理

### 9.1 总仓位规则

| 市场状态 | 最大仓位 |
|---|---|
| 强能量期 | 70% - 100% |
| 可交易期 | 30% - 60% |
| 混沌期 | 0% - 30% |
| 退潮期 | 0% |

### 9.2 单票仓位规则

| 标的类型 | 单票最大仓位 |
|---|---|
| 主线龙头 | 30% |
| 主线前排 | 20% |
| 低位补涨 | 10% - 15% |
| 新题材试错 | 5% - 10% |

### 9.3 禁止事项

- 禁止在退潮期满仓。
- 禁止在高潮末端重仓后排。
- 禁止亏损加仓摊低成本。
- 禁止在连续亏损后提高仓位。

---

## 10. 风控规则

### 10.1 单票止损

```text
单票最大亏损：-5% 到 -8%
若开盘直接低于预期且无承接，优先卖出。
若市场退潮，不等待技术反弹。
```

### 10.2 单日止损

```text
当日账户亏损超过 -3%，停止开新仓。
```

### 10.3 周度止损

```text
单周账户亏损超过 -8%，降低仓位或暂停交易。
```

### 10.4 连续亏损保护

```text
连续3笔亏损后，下一笔仓位减半。
连续5笔亏损后，暂停交易3个交易日。
```

### 10.5 黑名单机制

加入黑名单的情况：

1. 频繁炸板。
2. 高开低走严重。
3. 股性差，历史涨停无溢价。
4. 大股东减持、监管风险、财务异常。
5. 龙虎榜显示明显庄股特征。

## 11. 回测要求

### 11.1 数据要求

需要至少包括：

1. 日线行情数据。
2. 分钟线行情数据，建议1分钟级别。
3. 涨停/跌停价格数据。
4. 板块和概念归属数据。
5. 成交额、换手率、量比数据。
6. 涨停原因数据，若有更好。
7. 复权因子。
8. 停牌、退市、ST状态数据。

### 11.2 回测周期

建议覆盖：

```text
2019年至今
```

必须单独评估不同市场环境：

1. 牛市阶段。
2. 熊市阶段。
3. 震荡市阶段。
4. 题材活跃阶段。
5. 退潮杀跌阶段。

### 11.3 回测指标

必须输出：

| 指标 | 含义 |
|---|---|
| annual_return | 年化收益率 |
| max_drawdown | 最大回撤 |
| win_rate | 胜率 |
| profit_loss_ratio | 盈亏比 |
| sharpe_ratio | 夏普比率 |
| trade_count | 交易次数 |
| avg_holding_days | 平均持仓天数 |
| avg_profit_per_trade | 单笔平均收益 |
| max_single_loss | 单笔最大亏损 |
| monthly_return | 月度收益分布 |
| strategy_capacity | 策略容量 |

### 11.4 必须考虑的真实交易限制

1. 涨停无法买入。
2. 跌停无法卖出。
3. 一字板无法成交。
4. 滑点。
5. 佣金和印花税。
6. T+1限制。
7. 停牌风险。
8. 集合竞价价格不可完全预测。
9. 实盘成交量限制。

## 13. 策略伪代码

```python
for date in trading_days:
    market_score = calculate_market_energy(date)

    if market_score < 40:
        close_all_weak_positions()
        continue

    sectors = calculate_sector_energy(date)
    top_sectors = select_top_sectors(sectors, top_n=3)

    candidates = []

    for sector in top_sectors:
        sector_stage = identify_sector_stage(sector, date)

        if sector_stage in ["decline", "late_climax"]:
            continue

        stocks = get_sector_stocks(sector)

        for stock in stocks:
            if not pass_basic_filter(stock, date):
                continue

            stock_score = calculate_stock_energy(stock, date)
            leader_score = calculate_leader_score(stock, sector, date)

            if stock_score >= 70 and leader_score >= 60:
                candidates.append({
                    "stock": stock,
                    "sector": sector,
                    "stock_score": stock_score,
                    "leader_score": leader_score,
                    "sector_stage": sector_stage
                })

    candidates = sort_by_score(candidates)

    for position in current_positions:
        if trigger_sell_signal(position, date):
            sell(position)

    available_cash = calculate_position_limit(market_score)

    for candidate in candidates:
        if available_cash <= 0:
            break

        if trigger_buy_signal(candidate, date):
            buy(candidate, position_size=calculate_position_size(candidate, market_score))
```

## 16. 最重要的开发原则

### 16.1 不要只做选股器

本策略不是简单找强势股，而是完整判断：

```text
市场环境 → 题材方向 → 龙头强度 → 个股买点 → 仓位风控
```

如果只做个股强弱排名，很容易在退潮期买到高位下杀股。

### 16.2 必须先判断市场是否有赚钱效应

超短线最重要的不是进攻，而是判断能不能进攻。

市场没有赚钱效应时，最优策略是空仓。

### 16.3 必须区分前排和后排

同一个题材里，龙头和后排的风险收益完全不同。

优先级永远是：

```text
龙头 > 前排 > 补涨 > 后排
```

### 16.4 必须识别高潮和衰竭

超短线最大亏损通常来自：

```text
板块高潮后追后排
龙头断板后接补涨
市场退潮期强行打板
```

因此策略必须有退潮识别和空仓机制。
