# 戒定慧三学

## 概念定义

戒定慧是佛教修行的三大训练：戒是行为和关系的清净，定是注意力和心的稳定，慧是对无常、苦、无我、缘起的洞察。

| 三学 | 训练对象 | 核心作用 |
| --- | --- | --- |
| 戒 | 行为、语言、生活方式 | 减少混乱、伤害和后悔 |
| 定 | 注意力、稳定性、专注 | 让心能清楚观察 |
| 慧 | 认知、洞察、正见 | 看清执取和痛苦因缘 |

## 三学不是线性等级

| 关系 | 解释 |
| --- | --- |
| 戒支持定 | 行为混乱、谎言和伤害会制造内心扰动 |
| 定支持慧 | 心太散乱时，很难看清微细经验 |
| 慧支持戒 | 看清因果后，更愿意减少伤害性行为 |

## 戒：不是禁欲主义

戒的核心不是“别人规定我不能做什么”，而是建立一个减少后悔和伤害的生活结构。

| 戒的功能 | 例子 |
| --- | --- |
| 减少外部冲突 | 不欺骗、不伤害、不滥用语言 |
| 减少内部扰动 | 不做让自己反复愧疚的事 |
| 保护注意力 | 减少成瘾性刺激和过度放纵 |
| 建立可信关系 | 让修行不脱离现实伦理 |

## 定：不是发呆

定是稳定、持续、可调度的注意力。它不是昏沉，也不是逃避现实。

## 慧：不是懂很多术语

慧是直接看见经验的条件性和无常性。懂“无我”这个词不等于有无我智慧；能在自尊受伤时看见五蕴和执取，才是修行中的慧。

## 对个人成长的用法

| 问题 | 戒定慧视角 |
| --- | --- |
| 总是冲动 | 先用戒设置边界，再用定观察冲动 |
| 总是焦虑 | 用定稳定身心，用慧观察对未来的执取 |
| 总是自责 | 用慧看见自我叙事，用戒做可补救行动 |
| 总是拖延 | 用戒重建行为环境，用定训练启动能力 |
