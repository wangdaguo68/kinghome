# 飞书Webhook通知使用说明

## 功能说明

当回测系统检测到买入信号时，自动将股票代码和价格推送到飞书群聊。

## 获取飞书Webhook地址

1. 在飞书群聊中，点击右上角 `...` → `设置` → `群机器人`
2. 点击 `添加机器人` → `自定义机器人`
3. 设置机器人名称（如：量化交易助手）
4. 复制生成的 Webhook 地址（格式：`https://open.feishu.cn/open-apis/bot/v2/hook/xxxxx`）

## 使用方法

### 方式1：通过主程序菜单

```bash
python main.py
```

选择选项 `4. 扫描买入信号并推送飞书通知`，然后输入你的 Webhook 地址。

### 方式2：直接运行监控模块

编辑 `monitor.py`，将 `WEBHOOK_URL` 替换为你的实际地址：

```python
WEBHOOK_URL = "https://open.feishu.cn/open-apis/bot/v2/hook/你的webhook密钥"
```

然后运行：

```bash
python monitor.py
```

### 方式3：在代码中集成

```python
from monitor import scan_buy_signals

webhook_url = "https://open.feishu.cn/open-apis/bot/v2/hook/你的webhook密钥"
signals = scan_buy_signals(webhook_url=webhook_url)
```

## 通知消息格式

```
📈 买入信号提醒

股票代码: 600000
买入价格: ¥12.50
信号日期: 2026-03-12

请注意风险,仅供参考!
```

## 定时任务（可选）

如需每日自动扫描，可使用系统定时任务：

**Windows (任务计划程序)**
```
创建基本任务 → 每日触发 → 运行 python monitor.py
```

**macOS/Linux (crontab)**
```bash
# 每天收盘后15:30执行
30 15 * * 1-5 cd /path/to/quant-trading && python monitor.py
```

## 注意事项

- Webhook 地址请妥善保管，不要泄露
- 飞书机器人有频率限制，建议合理控制推送频率
- 信号仅供参考，投资需谨慎
