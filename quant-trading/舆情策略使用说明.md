# 舆情监控+大单跟随策略使用说明

## 功能特点

这是一个结合舆情分析、资金流向监控和机器学习的智能交易策略：

1. **多源舆情监控** - 实时抓取东方财富、财联社等平台新闻
2. **大单异动检测** - 监控主力资金净流入，捕捉大单点火信号
3. **机器学习预测** - 学习历史成功案例，预测交易时机
4. **自动化提醒** - 结合飞书通知，及时推送交易机会

## 快速开始

### 1. 实时监控

```python
from sentiment_strategy import SentimentStrategy

strategy = SentimentStrategy()

# 监控股票池
stock_pool = ['000001', '600519', '000858']
signals = strategy.monitor_and_alert(stock_pool)

for signal in signals:
    print(f"{signal['symbol']}: {signal['action']}")
    print(f"原因: {signal['reason']}")
```

### 2. 训练个性化模型

准备训练数据 `sentiment_training.xlsx`：

| symbol | sentiment_score | main_inflow_pct | ma5 | ma20 | volume_ratio | price_change | label |
|--------|----------------|-----------------|-----|------|--------------|--------------|-------|
| 000001 | 3 | 8.5 | 15.2 | 15.0 | 2.3 | 0.02 | 1 |
| 600519 | -2 | -3.2 | 1680 | 1690 | 0.8 | -0.01 | 0 |

label: 1=买入成功, 0=观望/失败

```python
# 训练模型
training_data = pd.read_excel('sentiment_training.xlsx')
strategy.train_model(training_data)
```

## 核心逻辑

1. **舆情评分** - 分析新闻标题和内容，计算正负面情绪得分
2. **大单触发** - 主力净流入超过5%视为大单点火
3. **技术验证** - 结合均线、成交量等技术指标
4. **ML决策** - 机器学习模型综合判断是否跟随

## 数据源

- 东方财富新闻 (ak.stock_news_em)
- 财联社电报 (ak.stock_telegraph_cls)
- 个股资金流 (ak.stock_individual_fund_flow_rank)
- 历史行情数据 (ak.stock_zh_a_hist)

## 注意事项

- 大单跟随有风险，建议小仓位试探
- 结合止损止盈策略使用
- 定期用实际交易结果重新训练模型
- 避免追高，注意风险控制
