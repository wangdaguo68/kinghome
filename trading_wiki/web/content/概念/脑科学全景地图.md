# 脑科学全景地图

> [!warning] 使用边界
> 本知识库用于学习和认知沉淀，不用于诊断、治疗或替代医生建议。脑科学很多结论是概率性、模型性和情境依赖的，不能把单个脑区、单个神经递质或一篇研究直接解释成完整的人性。

## 一句话定义

脑科学，也叫神经科学，是研究神经系统如何产生感觉、运动、情绪、记忆、注意、意识、决策和行为的学科。它连接生物学、心理学、医学、计算科学和哲学。

最有用的理解方式不是背脑区名字，而是建立一张多层地图：

1. **硬件层**：[[神经元突触与动作电位]]、神经胶质、脑区和神经通路。
2. **信号层**：[[神经递质与神经调质]]、电信号、化学信号和网络振荡。
3. **系统层**：[[大脑结构与功能定位]]、感觉、运动、情绪、记忆、注意和执行控制。
4. **功能层**：[[注意力系统与执行控制]]、[[记忆系统与海马体]]、[[奖赏系统与多巴胺]]、[[情绪系统与杏仁核]]。
5. **状态层**：[[睡眠与大脑清理修复]]、[[压力系统与HPA轴]]、疲劳、唤醒和恢复。
6. **身体行动层**：[[感知系统与身体内感]]、[[运动系统与小脑]]。
7. **改变层**：[[神经可塑性与学习]]、习惯、训练、环境和长期经验。
8. **自我与社会层**：[[默认模式网络与自我叙事]]、[[意识预测加工与自由意志]]、[[社会脑共情与镜像系统]]。

## 核心原则

| 原则 | 解释 | 对个人成长的启发 |
| --- | --- | --- |
| 大脑是预测机器 | 大脑不是被动接收世界，而是不断预测、校正和更新模型 | 改变行为，往往要改变预期和反馈 |
| 大脑是身体的一部分 | 情绪、睡眠、压力、代谢和免疫会影响认知 | 管理状态比单纯讲道理更基础 |
| 功能是网络产物 | 很少有单一脑区独立完成复杂心理功能 | 不要迷信“某个脑区负责某件事” |
| 学习依赖可塑性 | 重复、反馈、情绪和睡眠会改变连接强度 | 长期训练比短期刺激更可靠 |
| 奖赏塑造行为 | 预期奖励、即时反馈和环境线索会驱动习惯 | 设计环境比硬扛意志力更有效 |
| 情绪参与决策 | 情绪不是错误信号，而是价值、风险和身体状态的压缩表达 | 高压时不要做重大判断 |
| 自我叙事会反复运行 | 默认模式网络常参与回忆、想象和自我相关思考 | 觉察反刍能减少内耗 |

## 认知沉淀模板

以后写脑科学笔记，建议固定回答：

1. 这个概念解决什么问题？
2. 主要发生在哪些脑区、回路或状态中？
3. 它是事实、模型、假说还是流行解释？
4. 对注意力、情绪、学习、习惯或自我理解有什么影响？
5. 有哪些常见误解？
6. 我能如何在生活中观察或验证？

## 常见误区

| 误区 | 更准确的理解 |
| --- | --- |
| 左脑理性、右脑感性 | 左右半球有分工，但复杂认知依赖双侧网络协作 |
| 多巴胺等于快乐 | 多巴胺更常和动机、学习、奖励预测误差、行动选择有关 |
| 杏仁核等于恐惧中心 | 杏仁核参与威胁、显著性、情绪学习，不只处理恐惧 |
| 海马体就是记忆仓库 | 海马更像新记忆编码、情境绑定和提取索引系统 |
| 前额叶等于意志力 | 前额叶参与控制、计划、抑制和工作记忆，但会受压力、睡眠、情绪影响 |
| 神经可塑性意味着无限改变 | 可塑性存在边界，年龄、训练、反馈、睡眠和环境都会影响改变速度 |
| 脑区点亮说明心理原因 | fMRI 是间接测量血氧变化，不能单独证明因果 |
