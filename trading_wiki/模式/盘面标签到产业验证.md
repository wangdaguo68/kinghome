# 盘面标签到产业验证

## 模式定义

涨停复盘经常给公司贴上“PCB、WF6、PPE、InP”等标签。这个模式要求把标签拆成产业事实，再验证公司是否真实承接。

## 处理流程

| 步骤 | 动作 |
| --- | --- |
| 标签识别 | 记录涨停复盘中的标签 |
| 产业拆分 | 标签对应产品、工艺、材料还是设备 |
| 公司核验 | 年报、互动平台、公告、客户、订单 |
| 权限判断 | 没有事实验证则保持 observe_only |

## 典型案例

- [[宿迁联盛]]：磷化铟衬底 + 光稳定剂，需要确认是否真实有 InP。
- [[中化国际]]：PPE 树脂，需要确认产品和高速 PCB 关系。
- [[海亮股份]]：PCB + 铜箔，需要确认是否有 PCB 铜箔业务。

## 反例

- 只因为涨停摘要有标签就建核心标的。
- 不查主营业务和收入占比。
