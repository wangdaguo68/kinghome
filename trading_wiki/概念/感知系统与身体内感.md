# 感知系统与身体内感

## 概念定义

感知是大脑把外部刺激和内部身体信号组织成可理解体验的过程。身体内感 interoception 是对心跳、呼吸、胃肠、疼痛、紧张、疲劳等内部身体状态的感知。

易懂理解：大脑不只是看世界，也在不断读取身体；很多情绪和直觉都带有身体信号。

## 外感与内感

| 类型 | 信息来源 | 例子 |
| --- | --- | --- |
| 视觉 | 光信号 | 颜色、形状、运动 |
| 听觉 | 声波 | 语言、节奏、方向 |
| 触觉 | 皮肤和压力 | 温度、疼痛、接触 |
| 本体感觉 | 肌肉和关节 | 身体位置、动作感 |
| 前庭觉 | 内耳 | 平衡、加速度 |
| 身体内感 | 内脏和身体状态 | 心跳、呼吸、胃紧、疲惫 |

## 感知不是摄像

大脑会用过去经验和当前目标解释输入。因此同一刺激在不同情境下会有不同体验。

| 现象 | 解释 |
| --- | --- |
| 错觉 | 大脑根据常见规律做快速推断 |
| 注意影响感知 | 被注意的信息更容易被放大 |
| 情绪影响感知 | 威胁状态下更容易看到危险线索 |
| 身体状态影响判断 | 疲惫、心跳、饥饿会改变主观评价 |

## 岛叶与身体感

岛叶常被认为参与身体内部状态感知、疼痛、厌恶、情绪体验和显著性加工。它不是单一功能区，而是连接身体信号、情绪和认知的重要节点。

这解释了为什么焦虑常常不只是“想法”，还表现为胸闷、胃紧、心跳快、呼吸浅。

## 对个人成长的意义

| 练习 | 作用 |
| --- | --- |
| 身体扫描 | 提高对身体信号的觉察 |
| 呼吸观察 | 建立注意和自主神经调节入口 |
| 情绪命名 | 把身体感和情绪标签连接起来 |
| 延迟反应 | 不把身体紧张立刻解释成外部危险 |
| 睡眠和运动 | 改善身体底层状态 |

## 常见误区

| 误区 | 更准确说法 |
| --- | --- |
| 感知就是客观现实 | 感知是输入和大脑解释的结合 |
| 直觉一定可靠 | 直觉可能包含经验，也可能包含压力和偏见 |
| 情绪只是脑中的想法 | 情绪包含身体状态和行动准备 |
| 身体感受不重要 | 身体信号会影响注意、情绪和决策 |
