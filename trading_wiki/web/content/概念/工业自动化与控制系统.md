# 工业自动化与控制系统

## 一句话解释

工业自动化是用传感器、控制器、执行器、软件和设备把生产流程稳定、高效、可监控地运行起来。

## 基本结构

| 层级 | 作用 |
| --- | --- |
| 传感层 | 采集温度、压力、位置、视觉等数据 |
| 控制层 | PLC、运动控制器、工业电脑 |
| 执行层 | 电机、阀门、机械臂、产线设备 |
| 软件层 | MES、SCADA、质量系统 |
| 决策层 | 排产、优化、预测维护 |

## AI进入工业的路径

- 视觉质检。
- 预测性维护。
- 工艺参数优化。
- 机器人调度。
- 产线数据分析。

## 我的启发

工业 AI 的难点不是模型 Demo，而是数据质量、设备接口、现场稳定性、责任边界和投资回报。
