# 医学证据等级与健康决策

## 一句话解释

医学证据等级用于判断健康信息的可信度。个案和专家经验可以提供线索，但临床决策更重视系统综述、随机对照试验、指南和真实世界安全性。

## 常见证据层级

| 证据 | 强度 | 注意点 |
| --- | --- | --- |
| 个人体验 | 低 | 容易受安慰剂和选择偏差影响 |
| 个案报告 | 低 | 只能提示可能性 |
| 观察研究 | 中 | 相关不等于因果 |
| 随机对照试验 | 高 | 看样本、终点和适用人群 |
| 系统综述/Meta分析 | 较高 | 取决于纳入研究质量 |
| 临床指南 | 实用 | 需看更新日期和适用范围 |

## 健康决策问题

1. 这个建议针对什么人群？
2. 证据强度是什么？
3. 收益和风险分别是什么？
4. 是否有替代方案？
5. 是否需要医生参与？

## 我的启发

健康领域最容易被“单一案例 + 强情绪 + 商业转化”误导。知识库必须标注证据等级和适用边界。
