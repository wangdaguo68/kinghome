# 消费电子与端侧AI全景

## 概念定义

消费电子与端侧AI，指手机、PC、平板、耳机、手表、眼镜、车机和各类智能硬件在本地设备上运行 AI 模型、AI 助手和多模态交互能力的产业方向。

它不是简单的“给硬件加 AI 标签”。真正的端侧AI需要同时满足：芯片算力够、内存和存储够、系统级模型调度稳定、应用场景高频、隐私和延迟有优势，并且用户愿意为体验升级或换机付费。

## 产业链结构

| 环节 | 核心内容 | 国内外核心企业 | 验证重点 |
| --- | --- | --- | --- |
| 终端品牌 | 手机、PC、平板、可穿戴、眼镜 | Apple、Samsung、Huawei、小米、OPPO、vivo、联想、Dell、HP、华硕 | AI功能是否成为换机理由，出货量和高端机占比 |
| 主芯片/SoC | CPU、GPU、NPU、ISP、基带 | Apple Silicon、Qualcomm、MediaTek、Samsung、Intel、AMD、华为海思 | NPU TOPS、功耗、模型运行效率、生态适配 |
| 存储 | DRAM、LPDDR、NAND、UFS、SSD | Samsung、SK hynix、Micron、长江存储、兆易创新、江波龙、佰维存储 | AI功能是否提高内存和存储规格 |
| 传感器 | 摄像头、麦克风、IMU、生物识别 | Sony、Samsung、豪威集团、舜宇光学、丘钛科技、瑞声科技、歌尔股份 | 多模态输入是否提升摄像头、声学和传感器价值量 |
| 散热与结构件 | VC、石墨、导热材料、金属/玻璃/折叠结构 | 飞荣达、中石科技、领益智造、东山精密、蓝思科技、长盈精密 | 本地推理功耗是否推高散热和结构件要求 |
| 系统软件 | OS、AI助手、模型压缩、端云协同 | Apple、Google、Microsoft、Huawei、小米、荣耀、联想 | 是否形成跨应用调用和持续使用 |
| 应用层 | 影像、办公、翻译、搜索、日程、健康 | Microsoft、Adobe、Google、字节、腾讯、金山办公、美图、科大讯飞 | 是否产生付费、留存或生产力提升 |

## 需求端逻辑

| 场景 | 正面逻辑 | 反证 |
| --- | --- | --- |
| AI手机 | 语音助手、多模态搜索、影像编辑、隐私本地处理 | 用户感知弱、功能同质化、换机周期不缩短 |
| AI PC | 本地Copilot、办公自动化、会议总结、隐私数据处理 | 企业部署慢、软件生态不足、价格过高 |
| AI眼镜 | 第一视角拍摄、语音交互、实时翻译、导航 | 续航、重量、隐私、内容生态不足 |
| 可穿戴 | 健康监测、运动建议、情绪和睡眠分析 | 传感器准确性不足、医疗合规边界不清 |
| 智能家居 | 语音控制、场景自动化、家庭Agent | 设备割裂、隐私担忧、低频刚需 |

## 核心受益标的

只把“价值量提升或格局改善”相对明确的环节放进核心观察，不把所有消费电子公司都泛化成端侧AI标的。

| 标的/公司 | 位置 | 正面逻辑 | 需要验证 |
| --- | --- | --- | --- |
| Qualcomm | 手机与PC端AI芯片平台 | 安卓端侧AI和AI PC平台核心供应商 | 终端出货、OEM采用、NPU生态 |
| MediaTek | 安卓手机SoC | 中高端手机AI能力升级受益 | 高端份额和AI模型适配 |
| Apple | 端侧生态与软硬件一体化 | 芯片、系统、设备和隐私闭环强 | AI功能是否提升换机和服务收入 |
| 华为产业链 | 手机、PC、鸿蒙生态 | 国产软硬件生态协同 | 终端销量、鸿蒙应用、供应链订单 |
| 联想 | AI PC终端 | 企业和个人PC换机承接方 | AI PC渗透率、毛利和库存 |
| 豪威集团 | CMOS图像传感器 | 多模态输入提升摄像头规格 | 高端CIS导入和ASP |
| 舜宇光学/丘钛科技 | 光学模组 | AI影像和多摄方案受益 | 出货量、单机价值量和客户结构 |
| 歌尔股份/瑞声科技 | 声学、可穿戴、XR | AI语音、耳机、眼镜和XR交互入口 | 大客户订单和新品放量 |
| 飞荣达/中石科技 | 散热材料和模组 | 本地推理提高热管理要求 | 是否进入主流终端BOM |
| 江波龙/佰维存储 | 存储模组 | AI终端提高内存和存储规格 | 价格、订单和库存周期 |

## 关键验证清单

| 验证项 | 为什么重要 |
| --- | --- |
| 端侧模型是否高频使用 | 决定AI是不是换机理由 |
| 终端BOM是否真实升级 | 决定供应链价值量 |
| NPU、内存、散热和传感器规格是否提升 | 决定上游受益环节 |
| 大客户订单是否进入量产 | 决定公司业绩兑现 |
| 软件生态是否跨应用调用 | 决定端侧AI是否从功能变成平台 |
| 用户是否愿意付费或换机 | 决定产业链持续性 |

## 风险和误区

- 把发布会功能当作真实需求。
- 把“AI手机/AI PC”标签泛化到所有消费电子公司。
- 忽略硬件升级可能被整机厂压价。
- 忽略端云协同：很多功能不一定完全在本地运行。
- 忽略换机周期、宏观消费和库存周期。
