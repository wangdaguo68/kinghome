# 止观禅修与正念

## 概念定义

止和观是佛教禅修的两类核心训练。止偏向稳定和安住，观偏向洞察和看清。正念是清楚记得、持续知道当下经验的能力。

| 概念 | 简明解释 | 重点 |
| --- | --- | --- |
| 止 | 让心稳定在对象上 | 专注、安定、少散乱 |
| 观 | 观察经验的真实特征 | 无常、苦、无我、缘起 |
| 正念 | 清楚知道正在发生什么 | 不忘失、不被卷走 |
| 正定 | 更稳定成熟的定力 | 支持深度观察 |

## 正念不是简单放松

| 流行理解 | 佛学中的补充 |
| --- | --- |
| 正念等于放松 | 放松可能发生，但不是目标本身 |
| 正念等于专注呼吸 | 呼吸只是常用对象 |
| 正念等于不思考 | 念头出现也可以被观察 |
| 正念等于接纳一切 | 接纳不是纵容伤害行为 |

## 四念处

| 念处 | 观察对象 | 例子 |
| --- | --- | --- |
| 身念处 | 身体 | 呼吸、姿势、动作、身体感受 |
| 受念处 | 感受基调 | 苦受、乐受、不苦不乐受 |
| 心念处 | 心的状态 | 散乱、贪、嗔、昏沉、清明 |
| 法念处 | 心理规律和法类 | 五盖、五蕴、七觉支、四圣谛 |

## 一个安全的初学框架

1. 坐姿稳定，不追求特殊体验。
2. 以呼吸或身体触点为对象。
3. 分心时知道“分心了”，温和回来。
4. 情绪强烈时先回到身体，不急着解释。
5. 若出现强烈创伤反应、恐慌或失眠，应停止强练并寻求专业帮助。

## 禅修中的常见现象

| 现象 | 处理 |
| --- | --- |
| 念头很多 | 不是失败，知道念头多就是正念的一部分 |
| 昏沉困倦 | 调整姿势、睁眼、缩短时长 |
| 情绪浮现 | 回到身体感受，避免沉迷故事 |
| 想追求境界 | 观察“想要特殊体验”的贪 |
| 变得冷漠 | 检查是否缺少慈悲和伦理训练 |

## 与脑科学的连接

正念训练可以和[[注意力系统与执行控制]]、[[默认模式网络与自我叙事]]对话：它可能改变注意调度、情绪反应和自我叙事关系。但不同研究质量和训练类型差异很大，不能把“冥想改变大脑”当成万能结论。
