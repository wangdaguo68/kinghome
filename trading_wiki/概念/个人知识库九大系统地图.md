# 个人知识库九大系统地图

## 定位

本页是个人知识库的顶层架构。它解决的问题不是“今天新增哪篇笔记”，而是未来几年所有知识沉淀如何归位、互相连接、持续升级。

九大系统不是文件夹限制，而是思考入口。一个主题可以跨系统，例如 AI 医疗同时属于[[AI全产业链地图]]、[[医药全产业链地图]]、[[技术哲学与人类改变]]和生命系统。

## 放置规则

| 新知识类型 | 优先归入 | 例子 |
| --- | --- | --- |
| 关于思维模型、学习方法、判断框架 | 认知系统 | 归纳、演绎、模型、决策 |
| 关于 AI、模型、自动化 | 智能系统 | 大模型、Agent、具身智能 |
| 关于交易、公司、供应链、商业化 | 产业系统 | PCB、CPO、机器人零部件、产业复盘 |
| 关于资产、现金流和长期价值 | 财富系统 | 利率、通胀、资本结构 |
| 关于健康、医学、生物、心理和脑科学 | 生命系统 | 创新药、医疗器械、脑科学、心理学 |
| 关于材料、能源、制造 | 物理系统 | 半导体材料、设备、能源系统 |
| 关于审美、设计、体验 | 美学系统 | 品味、风格、界面、空间 |
| 关于哲学、文化、品牌、叙事 | 意义系统 | 哲学问题、品牌神话、文化符号、佛学 |
| 关于未来、范式、战略 | 范式系统 | Kuhn 范式、技术扩散、未来场景 |

## 系统之间的关键连接

| 连接 | 为什么重要 |
| --- | --- |
| 认知系统 -> 美学系统 | 审美首先是感知、注意、情绪和记忆的组织 |
| 美学系统 -> 意义系统 | 风格一旦稳定，就会变成符号、身份和文化 |
| 意义系统 -> 财富系统 | 品牌和叙事会影响定价权、估值和消费选择 |
| 智能系统 -> 产业系统 | AI 必须进入产品、流程和供应链才形成产业价值 |
| 物理系统 -> 产业系统 | 材料、能源、制造能力决定技术落地边界 |
| 范式系统 -> 全部系统 | 新范式会重新定义问题、方法、组织和价值 |

## 使用方法

1. 新主题先判断属于哪个系统。
2. 如果跨系统，主页面只放一个，其他系统用 wikilink 连接。
3. 每个系统先建立“全景地图”，再下钻概念页、案例页、人物页、方法页。
4. 非交易知识也要写清：定义、结构、误区、现实用法、来源。
5. 交易相关内容可以继续保留 `execution_permission`，非交易内容默认 `learning_only`。
