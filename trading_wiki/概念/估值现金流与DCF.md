# 估值现金流与DCF

## 一句话解释

估值本质上是在判断一项资产未来能产生多少现金流，以及这些现金流的不确定性应该打多少折扣。DCF 是把未来现金流折现到今天的估值方法。

## 核心变量

| 变量 | 影响 |
| --- | --- |
| 收入增长 | 决定未来规模 |
| 利润率 | 决定收入能转成多少利润 |
| 资本开支 | 决定增长是否消耗大量现金 |
| 折现率 | 反映资金成本和风险 |
| 永续增长 | 对长期估值影响很大 |
| 竞争格局 | 决定现金流能否持续 |

## DCF的风险

| 风险 | 说明 |
| --- | --- |
| 假设太敏感 | 小改长期增长率，估值变化巨大 |
| 过度精确 | 表格很精确，但输入可能很虚 |
| 忽略周期 | 把高景气当永续 |
| 忽略稀释 | 融资、股权激励会影响股东价值 |

## 我的启发

估值不是算出一个神奇数字，而是把市场隐含假设拆出来：它相信多高增长、多长时间、多高利润率、多低风险。投资判断要比较“市场假设”和“我的证据”。
