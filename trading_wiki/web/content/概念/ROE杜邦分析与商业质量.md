# ROE杜邦分析与商业质量

## 一句话解释

ROE 衡量股东权益能产生多少净利润。杜邦分析把 ROE 拆成净利率、资产周转率和杠杆，帮助看清公司赚钱靠什么。

## 杜邦拆解

ROE = 净利率 x 资产周转率 x 权益乘数

| 因素 | 含义 | 代表 |
| --- | --- | --- |
| 净利率 | 每一元收入赚多少钱 | 品牌、技术、定价权 |
| 资产周转率 | 资产使用效率 | 零售、制造、渠道效率 |
| 权益乘数 | 杠杆水平 | 金融、地产、重资产行业 |

## 高ROE的来源

| 来源 | 质量 |
| --- | --- |
| 强品牌和定价权 | 通常较高质量 |
| 技术壁垒和专利 | 取决于持续性 |
| 轻资产模式 | 可能高质量，也可能依赖外包 |
| 高杠杆 | 风险较高 |
| 周期高点利润 | 不可简单年化 |

## 我的启发

看 ROE 不要只看数字，要问它来自哪里、能持续多久、是否靠杠杆和周期。真正优秀的公司能在较长周期内维持较高资本回报。
