# 佛学全景地图

> [!warning] 使用边界
> 本知识库用于理解佛学思想、修行方法和自我观察，不用于替代医学、心理治疗或现实责任。佛学内部有南传、汉传、藏传以及现代学术研究等不同解释系统，遇到分歧时要标注传统来源和解释层级。

## 一句话定义

佛学是围绕“痛苦如何产生、如何被看清、如何被止息”的一套认识论、实践论和解脱论。它不只是宗教信仰，也包含关于经验、注意、欲望、自我感、情绪、伦理和训练方法的系统观察。

最核心的学习顺序是：

1. [[佛教基础与释迦牟尼]]：先知道佛教从哪里来，解决什么问题。
2. [[四圣谛与八正道]]：佛教最核心的诊断框架和实践路径。
3. [[缘起与十二因缘]]：理解“事物不是孤立存在”的因果网络。
4. [[三法印与无我]]：无常、苦、无我是观察经验的三把尺。
5. [[五蕴六处十八界]]：把“我”和“世界”拆成可观察的经验组件。
6. [[烦恼三毒与苦的生成]]：看清贪、嗔、痴如何制造痛苦。
7. [[戒定慧三学]]：从行为、注意力、洞察三层训练。
8. [[止观禅修与正念]]：具体练习如何落地。
9. [[业力轮回与解脱]]：理解行为、习气、后果和解脱的关系。
10. [[涅槃与中道]]：理解佛学不是压抑人生，而是超越两边执着。
11. [[空性与二谛]]：大乘佛学理解执着和真实的关键。
12. [[菩萨道与六度]]：从个人解脱扩展到慈悲和利他实践。
13. [[佛教宗派与经典地图]]：不同传统如何展开同一核心问题。
14. [[中国佛教禅宗净土天台华严]]：汉传佛教的主要脉络。
15. [[南传上座部与阿毗达摩]]：南传传统的修行和心理分析。
16. [[佛学与心理学脑科学连接]]：把佛学和现代认知科学连接，但不混同。
17. [[佛学常见误读与边界]]：避免把佛学变成鸡汤、虚无主义或逃避现实。

## 佛学的核心问题

| 问题 | 佛学回答方向 | 对个人知识库的价值 |
| --- | --- | --- |
| 我为什么反复被情绪拖走 | 因为触、受、爱、取形成自动链条 | 训练观察情绪发生的过程 |
| 为什么得到想要的东西仍然不安 | 欲望满足通常只暂时降低匮乏感 | 区分快乐、依赖和自由 |
| “我”到底是什么 | “我”是五蕴组合中被执取出来的身份感 | 减轻自我叙事的绝对化 |
| 怎样减少痛苦 | 不是压抑感受，而是看清因缘和执取 | 建立可执行的修行和复盘框架 |
| 慈悲为什么重要 | 他人痛苦和自身痛苦在因缘网络中相连 | 防止知识变成冷漠和自恋 |

## 三条学习主线

### 1. 理解主线

先掌握“四圣谛、缘起、三法印、五蕴”四个骨架。它们回答的是：痛苦是什么、从哪里来、经验如何构成、为什么没有一个固定不变的自我。

### 2. 实践主线

用“戒定慧”把理解落到行为。戒不是单纯道德禁令，而是减少后悔和混乱；定不是发呆，而是稳定注意；慧不是聪明，而是直接看清经验的无常、无我和执取。

### 3. 现代连接主线

把佛学和[[默认模式网络与自我叙事]]、[[注意力系统与执行控制]]、[[情绪系统与杏仁核]]连接时，只能说“有可对话的经验现象”，不能把古代修行术语直接等同于某个脑区或神经递质。

## 常见误区总表

| 误区 | 更准确的理解 |
| --- | --- |
| 佛学就是劝人看开 | 佛学强调看清因缘和执取，不是消极忍受 |
| 无我就是没有我、否定人格 | 无我是说没有固定、独立、永恒的实体我 |
| 空性就是一切都不存在 | 空性是否定自性，不是否定现象和因果 |
| 正念就是放松 | 正念是清楚知道当下经验，并不等于舒服 |
| 业力就是宿命论 | 业强调行为和结果的条件关系，不等于命中注定 |
| 慈悲就是讨好别人 | 慈悲包含智慧和边界，不是软弱依附 |
| 禅宗就是随心所欲 | 禅宗强调直指和实践，不是反智或任性 |
