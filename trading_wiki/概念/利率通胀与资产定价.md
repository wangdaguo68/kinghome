# 利率通胀与资产定价

## 一句话解释

利率是资金的价格，通胀是货币购买力和成本结构的变化。它们通过折现率、盈利预期、风险偏好和资金流向影响资产价格。

## 利率影响路径

| 路径 | 影响 |
| --- | --- |
| 折现率 | 利率上升，远期现金流估值承压 |
| 融资成本 | 高杠杆和成长公司压力更大 |
| 资产比较 | 债券收益率提高会改变股票吸引力 |
| 风险偏好 | 紧缩环境下市场更重视确定性 |

## 通胀影响路径

| 情况 | 可能影响 |
| --- | --- |
| 温和通胀 | 企业收入和名义利润可能提升 |
| 成本通胀 | 没有定价权的公司利润受压 |
| 高通胀 | 利率上升、估值压缩、需求受损 |
| 商品通胀 | 资源品和上游可能受益 |

## 我的启发

宏观不是每天解释涨跌的万能理由，而是市场风格的背景变量。利率、通胀、信用和财政组合不同，会改变成长、价值、周期、消费和科技的相对吸引力。
