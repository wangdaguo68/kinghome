# 烦恼三毒与苦的生成

## 概念定义

烦恼是扰动、污染和束缚身心的心理力量。三毒是最核心的烦恼根：贪、嗔、痴。

| 三毒 | 简明解释 | 日常表现 |
| --- | --- | --- |
| 贪 | 想抓住、占有、维持 | 上瘾、过度追求、害怕失去 |
| 嗔 | 排斥、敌意、抗拒 | 愤怒、怨恨、厌恶、攻击 |
| 痴 | 看不清事实和因缘 | 自欺、固执、把念头当事实 |

## 烦恼不是普通情绪本身

| 情绪 | 可能是正常反应 | 变成烦恼的标志 |
| --- | --- | --- |
| 愤怒 | 边界被侵犯 | 反复燃烧、想惩罚、失去判断 |
| 欲望 | 生命动力 | 不得到就痛苦，得到也不满足 |
| 恐惧 | 风险提醒 | 过度想象、回避所有不确定 |
| 悲伤 | 对失去的反应 | 固着于身份故事，无法流动 |

## 苦的生成链条

刺激 -> 感受 -> 解释 -> 贪嗔痴 -> 执取 -> 行动 -> 后果 -> 新习气

| 环节 | 交易或生活例子 |
| --- | --- |
| 刺激 | 看到别人赚钱、自己错过 |
| 感受 | 胸口发紧、焦虑 |
| 解释 | “我不行”“市场不等我” |
| 贪 | 想立刻补回来 |
| 嗔 | 讨厌自己、怪市场 |
| 痴 | 忘记风险和规则 |
| 执取 | 把结果等同于自我价值 |
| 行动 | 冲动决策 |
| 后果 | 亏损、后悔、强化焦虑 |

## 对治方向

| 烦恼 | 对治方向 | 对应笔记 |
| --- | --- | --- |
| 贪 | 布施、知足、观察欲望变化 | [[菩萨道与六度]]、[[三法印与无我]] |
| 嗔 | 慈悲、忍辱、观察受伤感 | [[菩萨道与六度]] |
| 痴 | 正见、学习、止观 | [[四圣谛与八正道]]、[[止观禅修与正念]] |

## 与现代心理学的连接

三毒可以和心理学中的冲动、回避、认知扭曲、情绪调节失效对话，也可以和[[奖赏系统与多巴胺]]、[[情绪系统与杏仁核]]连接。但三毒不是神经递质，也不是某个脑区。
